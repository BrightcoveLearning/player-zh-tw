樣式播放列表

在本主題中，您將學習如何設置播放列表的樣式。

概述

如果您希望更改播放列表的外觀，則可以使用CSS進行更改。在本文檔中，您將看到許多可以更改的樣式。當然，可能的更改很多，您需要找到正確的選擇器才能完成任務。

以下文檔是完成使用播放列表圖片的基本信息：
  • 實施播放列表：本文檔顯示了您需要對使用播放列表做出哪些決定，以及如何獲取將播放器代碼實際顯示在HTML頁面上所需的播放器代碼。
  • 播放列表用戶界面插件：本文檔詳細介紹了可用於增強播放列表功能外觀的選項。

設置垂直播放列表的樣式

下圖總結了垂直播放列表更改的樣式。該圖之後是一張列出樣式的表格。

控製播放列表上文字的顏色 .vjs-滑鼠右鍵 .vj-播放清單 {
  顏色：#E9D2B3;
}
控制的顏色
播放清單項目間的空格		 .vjs-滑鼠右鍵 .vj-播放清單 {
  背景顏色：#161D6E;
}
控制持續時間文字的顏色 .vjs-播放清單 .vj-播放清單-持續時間 {
  顏色：#9BB4DA;
}
控制持續時間背景的顏色 .vjs-播放清單 .vj-播放清單-持續時間 {
  背景顏色：#1160AA;
}
控製播放列表項之間的空間 .vjs-playlist-vertical .vjs-playlist-item {
  邊距頂部：20px;
}
為了防止在第一個播放列表項上方出現邊距，您應該遵循以下邊距設置樣式： 
.vjs-playlist-vertical .vjs-playlist-item:first-child {
  margin-top: 0px;
}

設置水平播放列表的樣式

下圖總結了為水平播放列表更改的樣式。該圖之後是一張列出樣式的表格。

控製播放列表上文字的顏色 .vjs-滑鼠右鍵 .vj-播放清單 {
  顏色：#E9D2B3;
}
控制的顏色
播放清單項目間的空格		 .vjs-滑鼠右鍵 .vj-播放清單 {
  背景顏色：#161D6E;
}
控制持續時間文字的顏色 .vjs-播放清單 .vj-播放清單-持續時間 {
  顏色：#9BB4DA;
}
控制持續時間背景的顏色 .vjs-播放清單 .vj-播放清單-持續時間 {
  背景顏色：#1160AA;
}
控製播放列表項之間的空間 .vjs-playlist-horizontal .vjs-playlist-item {
  左邊距：5px;
}
為防止第一個播放列表項左側出現邊距，您應遵循邊距設置樣式，如下所示： 
.vjs-playlist-horizontal .vjs-playlist-item:first-child {
  margin-left: 0px;
}

隱藏滾動條

不需要水平播放列表上的滾動條，但會出現。如果您想隱藏滾動條，則可以使用以下樣式：

div.vjs-playlist.vjs-playlist-horizontal {
  overflow-y: hidden;
}
