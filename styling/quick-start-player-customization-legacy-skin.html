快速開始：玩家自定義-舊版皮膚

本文件包含 Brightcove 播放器 1.x 外觀（石墨）的信息，而不是最新的外觀。向新皮膚的過渡發生在2015年12月的第二週。如需文件的最新版本，請參閱此文件。

在本主題中，提供了動手操作步驟，這些操作步驟可在以後自定義Brightcove Player的外觀。要自定義Brightcove Player的外觀，您需要使用級聯樣式表（CSS）。該文檔將經歷更改最初覆蓋視頻的播放按鈕的外觀和位置的過程。

概述

本文檔將分為幾節，內容介紹如何將CSS與頁面內嵌入一起使用，並且播放器的iframe實現是完全不同的。

本文檔中的內容使用Chrome的開發工具，但所有現代瀏覽器都具有等效的功能。有關在不同瀏覽器中使用開發工具的簡要介紹，請參見調試基礎文檔。

注意：Video Cloud Studio內部提供了一些基本的播放器樣式。如需詳細資訊，請參閱造型播放器。

確定玩家選擇器

CSS選擇器使您可以選擇和操作頁面上的HTML元素。CSS選擇器用於根據其ID，類，類型，屬性等來“查找"（或選擇）HTML元素。嘗試更改HTML頁面上元素的外觀時，您必須做的第一件事就是確定用於尋址元素的選擇器。以下步驟顯示瞭如何找到播放器的選擇器，然後更改播放器的大小。

  1. 在Video Cloud Studio中，打開“媒體"模塊，然後選擇要發布的視頻。
  2. 複製頁內發布代碼。
  3. 使用您的HTML編輯器，創建一個新的HTML頁面。
  4. 將發布代碼粘貼到HTML頁面的正文中。
  5. 瀏覽新創建的頁面。默認情況下，播放器的尺寸將為默認尺寸300 x 150像素。
  6. 打開瀏覽器的開發工具，然後深入HTML來查找div環繞播放器的標籤。您會看到div標籤具有一類video-js。這div包含播放器的播放器將始終具有video-js。video-js是在設計播放器樣式時需要的類名稱。
  7. 查看下面的屏幕截圖，以紅色突出顯示該課程。視頻標籤為Brightcove Player，以綠色突出顯示。

    注意：根據所使用的操作系統和瀏覽器，可能會顯示播放器的Flash後備版本。如果您的播放器使用的是Flash後備版本，則不會看到video標籤。相反，您會看到div圍繞object標籤，如下所示。

  8. 現在您知道了用於CSS選擇器的正確類，我們可以使用該類選擇器來更改播放器的大小。這style下方的方塊應放置在head HTML頁面的部分。在CSS術語中，這是使用類選擇器。 
            <style>
          .video-js {
            height: 344px;
            width: 610px;
          }
        </style>
      </head>
  9. 保存並瀏覽HTML頁面，您將看到播放器具有新的尺寸。

播放按鈕選擇器

將使用另一種方法來找到播放按鈕的選擇器。以下步驟顯示瞭如何為播放按鈕找到正確的選擇器，然後更改其外觀。

  1. 瀏覽您的HTML頁面。
  2. 右鍵單擊覆蓋播放器的播放按鈕，然後選擇檢查元素。
  3. 請注意div突出顯示的標記表示播放按鈕，並具有vjs-big-play-button。
  4. 更改樣式以包括CSS，該CSS會更改播放按鈕的顏色和不透明度（第17-18行），並使按鈕在播放器內部居中（第19-23行）。 
          <style>
          .video-js {
            height: 344px;
            width: 610px;
          }
          .vjs-big-play-button {
            background-color: red;
            opacity: .3;
            top: 0px;
            left: 0px;
            right: 0px;
            bottom: 0px;
            margin: auto;
          }
        </style>
  5. 瀏覽頁面，您將看到並非所有樣式都已使用。該按鈕不是紅色。
  6. 在開發人員工具的右側查看，您將看到樣式的摘要。您可以向下看，將背景更改為紅色被覆蓋了，以紅色表示出來。
    注意：在 Safari 中，即使樣式被覆蓋，也可能不會被刪除。

    發生的事情是對背景顏色進行了許多更改。您設置的樣式已在CSS的“層疊"部分中被覆蓋。您可以通過利用稱為CSS的功能來更改此設置特異性。特異性確定瀏覽器將應用哪個CSS規則，並且是一種權重類型，它與級聯樣式表（CSS）規則的顯示方式有關。使用特定性，可以使樣式比其他樣式更重要，從而強制使用它。有關CSS專用性的簡要介紹，請參見自定義玩家的外觀-舊版皮膚 doc，以及任何搜索中的大量參考文獻。

  7. 請注意div標籤周圍，播放器具有許多與之相關的類。
  8. 在這種情況下，請結合使用video-js和vjs-mouse將“增強"您的樣式優先。更改樣式以使用新的組合選擇器。請注意，前兩個類之間沒有空格，但最後一個類之前沒有空格（為什麼可以在特異性規則中找到詳細信息）。 
              .video-js.vjs-mouse .vjs-big-play-button {
            background-color: #ff0000;
            opacity: .3;
            top: 0px;
            left: 0px;
            right: 0px;
            bottom: 0px;
            margin: auto;
          }
  9. 瀏覽您的頁面，看到所有樣式現在都已應用。
  10. 再次查看開發人員工具的右側，您將看到樣式的摘要。現在，您將看到背景顏色更改沒有被覆蓋。

頁內嵌入

當使用上述選擇器和样式時，結果將與播放器的頁內嵌入實現一起使用，如下所示：

      <!doctype html>
      <html>
      
      <head>
          <meta charset="UTF-8">
          <title>Embed in page tester</title>
         <style>
          .video-js {
            height: 344px;
            width: 610px;
          }
          .video-js.vjs-mouse .vjs-big-play-button {
            background-color: red;
            opacity: .3;
            top: 0px;
            left: 0px;
            right: 0px;
            bottom: 0px;
            margin: auto;
          }
         </style>
      </head>
      <body>
      
        <video
          data-video-id="4443311217001"
          data-account="1507807800001"
          data-player="30a9c9e0-295b-4102-a7af-a3f86bba91ab"
          data-embed="default"
          class="video-js" controls></video>
        <script src="//players.brightcove.net/1507807800001/30a9c9e0-295b-4102-a7af-a3f86bba91ab_default/index.min.js"></script>
      
      
      </body>
      </html>

這是在HTML頁面上使用CSS的一種非常常見的方式。你看style區塊在頁面上video標籤。

注意：根據您的CSS經驗，您可能想知道為什麼上面的代碼沒有分配一個id歸因於video標籤。然後，您可以使用類似的選擇器#myPlayer .vjs-big-play-button。實際上，您可以擁有並且它會起作用。問題是，使用iframe播放器實現時，使用id選擇器創建的CSS無法使用，因此僅使用了現有的類名。通過不使用id屬性，樣式將直接適用於 iframe 播放器實現。

iframe

毫無疑問，iframe實現將Brightcove Player放在一個iframe中，然後將其放置在HTML頁面中。您可以使用包含iframe的頁面上的樣式來設置實際iframe本身的樣式，例如，更改其大小，但是iframe頁面上的樣式請勿層疊到iframe本身。iframe確實會拉入單獨的HTML頁面，因此，iframe內容的所需樣式必須與iframe中的HTML頁面相關聯。

使用Brightcove Player的iframe實現時，應用CSS的步驟為：

  • 將樣式放置在Internet上可訪問的文件中
  • 將樣式錶鍊接到播放器（使用Studio或播放器管理API完成）
注意：以下步驟假定您將使用上一步中完成的HTML進行工作，並顯示在上面的代碼片段中。如果您希望將其作為工作示例，只需將代碼複製到另一個文件中，然後在新文件中開始以下步驟。
  1. 在Studio中，發布視頻並複制代碼，以實現播放器的iframe實施。
  2. 在HTML頁面中，將頁面內嵌入代碼替換為iframe代碼。
  3. 改變.video-js使用一個元素選擇器的類選擇器iframe作為元素。離開width和height相同，如下所示： 
      iframe {
      height: 344px;
      width: 610px;
    }
  4. 檢查以確保您的HTML頁面看起來類似於以下內容： 
          <!doctype html>
      <html>
      <head>
        <meta charset="utf-8">
        <title>iframe tester</title>
      
         <style>
          iframe {
            height: 344px;
            width: 610px;
          }
          .video-js.vjs-mouse .vjs-big-play-button {
            background-color: red;
            opacity: .3;
            top: 0px;
            left: 0px;
            right: 0px;
            bottom: 0px;
            margin: auto;
          }
         </style>
      <body>
      
        <iframe src='//players.brightcove.net/1507807800001/30a9c9e0-295b-4102-a7af-a3f86bba91ab_default/index.html?videoId=4443311217001'
          allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>
      
      </body>
      </html>
  5. 瀏覽頁面，您將看到iframe播放器的尺寸正確。如前所述，按鈕的樣式將不起作用。
  6. 創建一個名為custom-qs.css並將其保存在Internet上可訪問的位置。
  7. 剪切按鈕的樣式並將其粘貼到文件中並保存。僅剪切按鈕的樣式，不能包含按鈕的樣式style標籤。該文件應僅包含以下內容： 
          .video-js.vjs-mouse .vjs-big-play-button {
        background-color: red;
        opacity: .3;
        top: 0px;
        left: 0px;
        right: 0px;
        bottom: 0px;
        margin: auto;
      }
  8. 檢查以確保您的HTML頁面看起來類似於以下內容： 
          <!doctype html>
      <html>
      <head>
        <meta charset="utf-8">
        <title>iframe tester</title>
      
        <style>
          iframe {
            height: 344px;
            width: 610px;
          }
        </style>
      <body>
      
        <iframe src='//players.brightcove.net/1507807800001/30a9c9e0-295b-4102-a7af-a3f86bba91ab_default/index.html?videoId=4443311217001'
          allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>
      
      </body>
      </html>
  9. 在Studio中，進入玩家們模塊並編輯用於此快速入門的播放器。
  10. 找到插件部分，然後點擊編輯按鈕。
  11. 點擊的CSS輸入並輸入CSS文件的路徑，如下所示：
  12. 點擊保存將您的更改保存在插件部分。
    頁面頂部的預覽播放器將更新以顯示應用於按鈕的 CSS 樣式。
  13. 在頁面頂部，單擊發布發布您的播放器。
  14. 返回瀏覽器並刷新頁面。啟用發布的更改後，您將看到您的樣式已應用於iframe播放器。可能要過幾分鐘才能看到發布的更改。
