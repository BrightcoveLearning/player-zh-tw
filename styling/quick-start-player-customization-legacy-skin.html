    快速入門：玩家自訂-傳統造型

    本文件包含 Brightcove 播放器 1.x 外觀（石墨）的信息，而不是最新的外觀。這種轉變到較新的皮膚發生在 2015 年 12 月的第二週。如需文件的最新版本，請參閱此文件。

    在本主題中，會提供實際操作步驟，以便隨後自訂 Brightcove 播放程式的外觀。要自定義布萊特灣播放器的外觀，您需要使用級聯樣式表（CSS）。本文件將會經過變更最初覆蓋視訊之播放按鈕的外觀和位置的程序。

    概覽

    本文件將有幾個部分，因為使用 CSS 與頁面內嵌和播放器的 iframe 實現完全不同。

    本文件中的內容使用 Chrome 的開發工具，但所有現代瀏覽器都具有同等功能。如需在不同瀏覽器中使用開發工具的簡要介紹，請參閱偵錯基礎知識文件。

    注意：一些基本的播放器樣式視頻雲工作室內提供。如需詳細資訊，請參閱造型播放器。

    決定播放器選擇器

    CSS 選擇器允許您選擇和操作頁面上的 HTML 元素。CSS 選擇器用於根據它們的 ID，類，類型，屬性等「查找」（或選擇）HTML 元素。嘗試改變 HTML 頁面上元素的外觀時，您必須做的第一件事是確定用於解決元素的選擇器。以下步驟顯示如何找到播放器的選擇器，然後更改播放器的大小。

    1. 在視訊雲端 Studio 中，開啟媒體模組，然後選取要發佈的視訊。
    2. 複製頁內發布代碼。
    3. 使用您的 HTML 編輯器，創建一個新的 HTML 頁面。
    4. 將發佈程式碼貼到 HTML 網頁內文中。
    5. 瀏覽新建立的頁面。默認情況下，播放器的尺寸將為默認尺寸300 x 150像素。
    6. 打開瀏覽器的開發工具，然後深入HTML來查找div環繞播放器的標籤。您會看到div標籤具有一類video-js。的div包含玩家的總會有一類video-js。video-js是在設計播放器樣式時需要的類名稱。
    7. 檢查下面的屏幕截圖，看到以紅色突出顯示的班級。視頻標記為Brightcove Player，以綠色突出顯示。

      注意：根據所使用的操作系統和瀏覽器，可能會顯示播放器的Flash後備版本。如果您的播放器使用的是Flash後備版本，則不會看到video標籤。相反，您會看到div圍繞object標籤，如下所示。

    8. 現在您知道了用於CSS選擇器的正確類，我們可以使用該類選擇器來更改播放器的大小。的style下面的塊應放置在head HTML頁面的部分。在CSS術語中，這是使用類選擇器。 
              <style>
          .video-js {
            height: 344px;
            width: 610px;
          }
        </style>
      </head>
    9. 保存並瀏覽HTML頁面，您將看到播放器具有新的尺寸。

    播放按鈕選擇器

    另一種方法將被用來找到播放按鈕的選擇器。下列步驟說明如何尋找播放按鈕的正確選擇器，然後變更其外觀。

    1. 瀏覽您的 HTML 網頁。
    2. 右鍵單擊覆蓋播放器的播放按鈕，然後選擇檢查元素。
    3. 請注意div突出顯示的標記表示播放按鈕，並具有vjs-big-play-button。
    4. 更改樣式以包括CSS，該CSS可以更改播放按鈕的顏色和不透明度（第17-18行），並使按鈕在播放器內部居中（第19-23行）。 
            <style>
          .video-js {
            height: 344px;
            width: 610px;
          }
          .vjs-big-play-button {
            background-color: red;
            opacity: .3;
            top: 0px;
            left: 0px;
            right: 0px;
            bottom: 0px;
            margin: auto;
          }
        </style>
    5. 瀏覽您的頁面，你會看到沒有所有的樣式已被使用。按鈕不是紅色的。
    6. 看看你的開發人員工具的右側，你會看到你的樣式的摘要。你可以往下看，看到將背景更改為紅色被覆蓋，由它被劃掉。
      注意：在 Safari 中，即使樣式被覆蓋，也可能不會被刪除。

      發生的事情是對背景顏色進行了許多變化。您設置的樣式已被覆蓋在 CSS 的「級聯」部分。您可以通過利用 CSS 稱為特異性的功能來改變這一點。特異性決定了哪些 CSS 規則是由瀏覽器應用，並且是一種加權，它與層疊樣式表（CSS）規則的顯示方式有關。使用特殊性，你可以使你的風格比其他人更重要，迫使它被使用。有關CSS專用性的簡要介紹，請參見自定義玩家外觀-舊版皮膚 doc，以及任何搜索中的大量參考文獻。

    7. 請注意div標籤周圍，播放器具有許多與之相關的類。
    8. 在這種情況下，請結合使用video-js和vjs-mouse將“增強”您的樣式優先。變更您的樣式以使用新的組合選取器。請注意，前兩個類之間沒有空格，但最後一個類之前沒有空格（為什麼可以在特異性規則中找到詳細信息）。 
                .video-js.vjs-mouse .vjs-big-play-button {
            background-color: #ff0000;
            opacity: .3;
            top: 0px;
            left: 0px;
            right: 0px;
            bottom: 0px;
            margin: auto;
          }
    9. 瀏覽您的頁面，看看現在已套用所有樣式。
    10. 再次看看你的開發人員工具的右側，你會看到你的樣式的摘要。現在你看到你的背景顏色更改沒有被覆蓋。

    頁內嵌

    當使用上述選擇器和样式時，結果將與播放器的頁面內嵌入實現一起使用，如下所示：

          <!doctype html>
      <html>
      
      <head>
          <meta charset="UTF-8">
          <title>Embed in page tester</title>
         <style>
          .video-js {
            height: 344px;
            width: 610px;
          }
          .video-js.vjs-mouse .vjs-big-play-button {
            background-color: red;
            opacity: .3;
            top: 0px;
            left: 0px;
            right: 0px;
            bottom: 0px;
            margin: auto;
          }
         </style>
      </head>
      <body>
      
        <video
          data-video-id="4443311217001"
          data-account="1507807800001"
          data-player="30a9c9e0-295b-4102-a7af-a3f86bba91ab"
          data-embed="default"
          class="video-js" controls></video>
        <script src="//players.brightcove.net/1507807800001/30a9c9e0-295b-4102-a7af-a3f86bba91ab_default/index.min.js"></script>
      
      
      </body>
      </html>

    這是使用 CSS 與 HTML 頁面的一個非常常見的方式。你看style區塊在頁面上video標籤。

    注意：根據您的CSS經驗，您可能想知道為什麼上面的代碼沒有分配一個id歸因於video標籤。然後你可以使用類似的選擇器#myPlayer .vjs-big-play-button。事實上，你可以擁有，它會起作用。問題是，使用 id 選擇器創建的 CSS 在使用 iframe 播放器實現時不起作用，因此只使用現有的類名。通過不使用id屬性，樣式將直接適用於 iframe 播放器實現。

    iframe

    毫無疑問，iframe實現將Brightcove Player放在一個iframe中，然後將其放置在HTML頁面中。您可以對實際的 iframe 本身進行樣式，例如使用包含 iframe 的頁面上的樣式更改其大小，但是 iframe 頁面上的樣式不要級聯到 iframe 本身。iframe 字面上提取一個單獨的 HTML 頁面，因此 iframe 內容的所需樣式必須與 iframe 中的 HTML 頁面關聯。

    當使用布萊特灣播放器的 iframe 實現時，應用 CSS 的步驟如下：

    • 在可存取網際網路的檔案中放置型式
    • 將樣式表連結至播放器 (使用 Studio 或播放程式管理 API 完成)
    注意：下面的步驟假設你將從你在上面的步驟中完成的 HTML 工作，並顯示在上面的代碼片段。如果您希望將其作為一個工作示例，只需將代碼複製到另一個文件中，並在新文件中啟動以下步驟即可。
    1. 在 Studio 中，發布視頻並複製播放器的 iframe 實現的代碼。
    2. 在您的 HTML 頁面中，將頁面內嵌程式碼取代為 iframe 程式碼。
    3. 改變.video-js使用一個元素選擇器的類選擇器iframe作為元素。離開width和height相同，如下所示： 
        iframe {
      height: 344px;
      width: 610px;
    }
    4. 請檢查以確定您的 HTML 網頁看起來類似下列內容： 
            <!doctype html>
      <html>
      <head>
        <meta charset="utf-8">
        <title>iframe tester</title>
      
         <style>
          iframe {
            height: 344px;
            width: 610px;
          }
          .video-js.vjs-mouse .vjs-big-play-button {
            background-color: red;
            opacity: .3;
            top: 0px;
            left: 0px;
            right: 0px;
            bottom: 0px;
            margin: auto;
          }
         </style>
      <body>
      
        <iframe src='//players.brightcove.net/1507807800001/30a9c9e0-295b-4102-a7af-a3f86bba91ab_default/index.html?videoId=4443311217001'
          allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>
      
      </body>
      </html>
    5. 瀏覽頁面，您將看到iframe播放器的尺寸正確。如前所述，按鈕的樣式將不起作用。
