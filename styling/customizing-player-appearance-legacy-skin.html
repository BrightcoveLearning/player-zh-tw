    自訂玩家外觀-舊版外觀

    本文件包含 Brightcove 播放器 1.x 外觀（石墨）的信息，而不是最新的外觀。這種轉變到較新的皮膚發生在 2015 年 12 月的第二週。如需文件的最新版本，請參閱此文件。

    本主題是自定義new / styling / customizing-player-appearance.html播放器外觀的入門。內容使用 Chrome 的開發工具，但所有現代瀏覽器都具有同等功能。有關在不同瀏覽器中使用開發工具的簡要介紹，請參見JavaScript調試基礎文件。

    建立自己的外觀

    如果您不希望使用預設外觀並建立您自己的外觀，則可以使用播放程式管理 API 進行此操作。播放程式組態指南提供必要的資訊。

    播放程式

    注意：本文件假設 CSS 的一些知識，並使用瀏覽器的開發工具來檢視 HTML 元素的類別，並檢查適用的 CSS。如果您不具備先決條件的基礎知識，則最好是通過快速開始：玩家自定義-舊版皮膚。

    您可以改變播放器的外觀，但您必須先有辦法解決它。要查看此內容，請瀏覽包含播放器embed_in_page代碼的HTML頁面。在裡面元素開發工具部分找到第一個<div>您將看到它具有一個帶有值的類video-js分配，等等。

    一旦你知道這一點，你可以使用樣式來改變玩家本身。例如，要更改播放器的大小，可以使用以下樣式：

    <style>
  .video-js {
    width: 640px;
    height: 360px;
  }
</style>

    框架播放器

    如果您使用播放器的iframe實現，則情況會有所不同。您仍然會看到帶有video-js類，但它當然會在iframe中。

    您創建的樣式仍然可以與 iframe 中的播放器一起使用，但需要創建 CSS 文件並與使用 Studio 的播放器相關聯。您可以通過轉到玩家們模塊，然後單擊要與CSS關聯的視頻，然後在外掛程式部分添加您的CSS文件的路徑。

    如果您確實希望自定義iframe本身，則可以通過添加一個class歸因於iframe元素（第27行）。在這種情況下，類值是videoFrame。然後使用<style>標籤根據需要進行修改。在下面的示例中width和height被改變了。

    <!doctype html>
<html>

<head>
  <meta charset="UTF-8">
  <title>Customize iframe</title>

  <!-- Page styles -->
  <style>
    .videoFrame {
      width: 640px;
      height: 360px;
    }
  </style>
</head>

<body>
  <iframe class="videoFrame" src='//players.brightcove.net/1752604059001/68f30408-8fb4-431d-accb-8c5baa8c4790_default/index.html' allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>
</body>

</html>

    播放按鈕

    如果你想改變播放按鈕的外觀，你首先需要知道如何解決它。在Chrome中，請確保開發工具打開，然後右鍵單擊該按鈕。從出現的選擇中選擇檢查元素。

    在裡面元素開發工具”部分，您將看到與按鈕元素相對應的HTML代碼。

    從上面突出顯示的代碼，你可以看到分配給按鈕的類是vjs-big-play-button。播放按鈕的顏色（與老虎相匹配）和不透明度將隨著以下樣式改變：

      <style>
    .vjs-big-play-button {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

    你會認為這種方法會起作用，但事實並非如此。使用 CSS 時，您需要知道屬性的特殊性。這是指瀏覽器如何確定哪些屬性值得根據它們的相關性應用到一個元素。

    應用按鈕更改的一種方法是在CSS選擇器中引用ID。這是因為id具有很高的CSS特異性。有關更多詳細信息，請參見CSS特異性部分

    首先，添加一個id歸因於div播放器代碼的元素。

        <video-js id="video_1"
      data-account="1752604059001"
      data-player="68f30408-8fb4-431d-accb-8c5baa8c4790"
      data-embed="default"
      class="video-js" controls></video-js>

    <script src="//players.brightcove.net/1752604059001/68f30408-8fb4-431d-accb-8c5baa8c4790_default/index.min.js"></script>

    然後，在 CSS 中引用玩家的大型播放按鈕，如下所示：

      <style>
    #video_1 .vjs-big-play-button {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

    中心播放按鈕

    進度和數量

    我們將在本文檔中查看的最後一個外觀定制將是更改進度和音量指示器的顏色。兩者的默認顏色均為紫紅色，如下所示：

    這裡的挑戰是找到元素的類名。使用與上述相同的技術，右鍵單擊元素選擇檢查元素，您可以轉到整個進度指示器，然後向下鑽取將看到vjs-play-progress類。

    在Chrome中元素面板，將鼠標懸停在div類名稱為的元素vjs-play-progress在播放器中查看以下內容：

    檢查音量級別元素時，Chrome中的工具非常有用。請注意，只有將鼠標懸停在音量按鈕上才能看到音量。要在檢查和查找正確的元素時顯示音量級別，請執行以下步驟：

    • 右鍵單擊音量按鈕並檢查
    • 點擊切換元素按鈕（下方以藍色突出顯示）款式部分
    • 檢查：徘徊框（在框中以紅色突出顯示款式部分）以在您檢查元素時模擬鼠標懸停
    • 向下鑽取，直到找到音量級別元素（在元素部分）

    現在background-color這兩個元素都需要進行樣式設置。您希望它們與按鈕具有相同的顏色，因此請如下所示更改樣式：

      <style>
    #video_1 .vjs-big-play-button {
      opacity: .6;
    }
    #video_1 .vjs-big-play-button,
    #video_1 .vjs-play-progress,
    #video_1 .vjs-volume-level {
      background-color: #B37D5B;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

    請注意，我們在每個類別的前面都加上了玩家的ID div元件。這為CSS特異性方面的類更改提供了相關性。這將在下一節中討論。

    該入門知識可讓您快速了解播放器的樣式。如您所見，方法是找到與您要更改的元素相對應的類名稱，然後設置其樣式。請記住，如iframe如果您使用的是iframe播放器，則必須將樣式放在文件中，然後使用curl語句將樣式與播放器相關聯。

    請記住，最佳做法是將自定義設置添加到播放器配置中，以便可以對其進行優化並隨播放器一起嵌入。

    控制可見性

    您可以指定控件可見的時間。您使用options().inactivityTimeout屬性來做到這一點。您將值設置為一個數字，該數字是在播放器上的任何活動結束後控件保持可見狀態的秒數。要使控件始終可見，請將其值設置為0。

    以下代碼顯示了設置inactivityTimeout至0，這將導致控件始終可見。

    <video-js id="myPlayerID"
  data-account="1752604059001"
  data-player="04ee9f58-9d08-43ea-803f-abda86776db2"
  data-embed="default"
  class="video-js" controls></video-js>
<script src="//players.brightcove.net/1752604059001/04ee9f58-9d08-43ea-803f-abda86776db2_default/index.min.js"></script>

<script>
  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;
    myPlayer.options().inactivityTimeout = 0;
  });
</script>

    控制欄圖標順序

    您可以使用以下命令控制控制欄中圖標的順序order樣式分配了一個數字。例如， 5將在進度條之後但在音量按鈕之前放置一個圖標，並且9會將圖標放置在控制欄的最右側。對於較舊的瀏覽器，可能需要更多樣式，如下所示：

    {
  order: 5; /* after the progress bar but before the volume button */
  -webkit-box-ordinal-group: 5;
  -moz-box-ordinal-group: 5;
  -webkit-order: 5;
  -ms-flex-order: 5;
}

    新增圖示的範例 (在這種情況下為下載檔案) 會顯示在 Brightcove 播放程式範例中：下載視訊外掛程式文件。

    CSS 特殊性

    CSS 特異性是指瀏覽器確定哪些屬性值根據其相關性應用於元素的方式。如需有關此概念的詳細資訊，請檢閱特殊性文章。

    在 Chrome 中，檢查大型播放按鈕。請注意，該background屬性是使用多個類別設定的。這意味著我們不能只使用一個.vjs-big-play-button類來覆蓋此屬性。

    以下是自訂大型播放按鈕的三種方法：

    • 參照分配給玩家的 ID。
    • 使用多個類別來提高屬性的相關性。
    • 使用 JavaScript 在播放器上設置 ID 屬性，然後在 CSS 中引用它。

    引用一個 ID

    由於id屬性與 CSS 特異性有很高的相關性，因此在 CSS 選擇器中id添加一個將覆蓋該background-color屬性。

    如果您尚未這樣做，請將id屬性添加到video元素，如下所示：

        <video-js id="video_1"
      data-account="1752604059001"
      data-player="68f30408-8fb4-431d-accb-8c5baa8c4790"
      data-embed="default"
      class="video-js" controls></video-js>

    在 CSS 中，將 id 值添加到.vjs-big-play-button類選擇器。取代背景顏色和不透明度，如下所示：

      <style>
    #video_1 .vjs-big-play-button {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

    你應該看到大的播放按鈕現在是橙色的。

    使用多個類別

    自從background大播放按鈕的屬性是使用兩個類設置的，因此在CSS選擇器中至少需要兩個類才能覆蓋此屬性。

    將類添加到.video-js.vjs-big-play-button類選擇器中。請記住，該.video-js類是在視頻容器上設置的。

    使用這些值，就是說選擇所有類名稱為vjs-big-play-button這是類名稱為的元素的後代.video-js。

    <style>
  .video-js .vjs-big-play-button {
    background-color: #B37D5B;
    opacity: .6;
  }
  .video-js {
    width: 640px;
    height: 360px;
  }
</style>

    你應該看到大的播放按鈕現在是橙色的。使用多個類可能非常棘手，可能會影響多個元素，而 id 是特定於一個元素。

    使用

    您也可以使用 JavaScript 動態添加一個 ID 到大型播放按鈕。然後你可以從 CSS 引用該 ID。

    添加代碼以添加一個 ID 到大播放按鈕元素。

    • 第 173 行：獲取對具有類的第一個元素的引用vjs-big-play-button。
    • 第 174 號線：添加一個id屬性到大播放按鈕。
      <script type="text/JavaScript">

    var playButton = document.getElementsByClassName("vjs-big-play-button")[0];
    playButton.setAttribute("id", "myPlayButton");

  </script>

    在 CSS 中，將選擇器更改為使用myPlayButton ID 名稱。

      <style>
    #myPlayButton {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

    你應該看到大的播放按鈕現在是橙色的。
