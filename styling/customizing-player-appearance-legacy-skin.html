自定義玩家外觀-舊版外觀

本文件包含 Brightcove 播放器 1.x 外觀（石墨）的信息，而不是最新的外觀。向新皮膚的過渡發生在2015年12月的第二週。如需文件的最新版本，請參閱此文件。

本主題是自定義new / styling / customizing-player-appearance.html播放器外觀的入門。內容使用Chrome的開發工具，但所有現代瀏覽器都具有等效的功能。有關在不同瀏覽器中使用開發工具的簡要介紹，請參見JavaScript調試基礎文檔。

創建自己的皮膚

如果您不想使用默認外觀並創建自己的外觀，則可以使用Player Management API來實現。這播放器配置指南提供必要的信息。

玩家

注意：本文檔假定您對CSS有一定的了解，並使用瀏覽器的開發工具來查看HTML元素的類並檢查適用的CSS。如果您不具備先決條件的基礎知識，則最好是通過快速開始：玩家定制 - 傳統皮膚.

您可以更改播放器的外觀，但是必須首先有一種解決它的方法。要查看此內容，請瀏覽包含播放器embed_in_page代碼的HTML頁面。在裡面元素開發工具部分找到第一個<div>您將看到它具有一個帶有值的類video-js分配，等等。

一旦知道這一點，就可以使用樣式來更改播放器本身。例如，要更改播放器的大小，可以使用以下樣式：

<style>
  .video-js {
    width: 640px;
    height: 360px;
  }
</style>

iframe播放器

如果您使用播放器的iframe實現，則情況會有所不同。您仍然會看到帶有video-js類，但是它當然會在iframe中。

您創建的樣式仍然可以在iframe中與播放器一起使用，但是需要使用Studio創建CSS文件並將其與播放器關聯。您可以通過轉到玩家們模塊，然後單擊要與CSS關聯的視頻，然後在外掛程式部分添加您的CSS文件的路徑。

如果您希望自定義iframe本身，則可以通過添加一個class歸因於iframe元素（第27行）。在這種情況下，類值是videoFrame。然後用<style>標籤進行修改。在下面的示例中width和height被改變了。

<!doctype html>
<html>

<head>
  <meta charset="UTF-8">
  <title>Customize iframe</title>

  <!-- Page styles -->
  <style>
    .videoFrame {
      width: 640px;
      height: 360px;
    }
  </style>
</head>

<body>
  <iframe class="videoFrame" src='//players.brightcove.net/1752604059001/68f30408-8fb4-431d-accb-8c5baa8c4790_default/index.html' allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>
</body>

</html>

播放按鈕

如果要更改播放按鈕的外觀，則首先需要知道如何解決它。在Chrome中，請確保開發工具打開，然後右鍵單擊該按鈕。從出現的選擇中選擇檢查元素。

在裡面元素開發工具"部分，您將看到與按鈕元素相對應的HTML代碼。

從上面突出顯示的代碼中，您可以看到分配給按鈕的類是vjs-big-play-button。播放按鈕的顏色（與老虎匹配）和不透明度將通過以下樣式進行更改：

  <style>
    .vjs-big-play-button {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

您可能會認為這種方法會起作用，但不會起作用。使用CSS時，您需要了解屬性的特殊性。這是指瀏覽器如何根據相關性確定哪些屬性值應用於元素。

應用按鈕更改的一種方法是在CSS選擇器中引用ID。這是因為id具有很高的CSS特異性。有關更多詳細信息，請參見CSS特異性部分

首先，添加一個id歸因於div播放器代碼的元素。

    <video-js id="video_1"
      data-account="1752604059001"
      data-player="68f30408-8fb4-431d-accb-8c5baa8c4790"
      data-embed="default"
      class="video-js" controls></video-js>

    <script src="//players.brightcove.net/1752604059001/68f30408-8fb4-431d-accb-8c5baa8c4790_default/index.min.js"></script>

然後，按以下方式在CSS中引用播放器的“大播放"按鈕：

  <style>
    #video_1 .vjs-big-play-button {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

這是將按鈕從深灰色更改為橙色的結果：

中心播放按鈕

您可以使用以下樣式在水平和垂直方向上將播放按鈕居中：

top: 0px;
left: 0px;
right: 0px;
bottom: 0px;
margin: auto;

隱藏播放按鈕

您可以使用以下樣式完全隱藏播放按鈕：

display: none;

    #video_1 .vjs-big-play-button {
      opacity: .6;

      width: 640px;
      height: 360px;
    }
  </style>

控制可見性

您可以指定控件可見的時間。您使用options().inactivityTimeout屬性來做到這一點。您將值設置為一個數字，該數字是在播放器上的任何活動結束後控件保持可見狀態的秒數。要使控件始終可見，請將其值設置為0。

以下代碼顯示了設置inactivityTimeout至0，這將使控件始終可見。

<video-js id="myPlayerID"
  data-account="1752604059001"
  data-player="04ee9f58-9d08-43ea-803f-abda86776db2"
  data-embed="default"
  class="video-js" controls></video-js>
<script src="//players.brightcove.net/1752604059001/04ee9f58-9d08-43ea-803f-abda86776db2_default/index.min.js"></script>

<script>
  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;
    myPlayer.options().inactivityTimeout = 0;
  });
</script>

控制欄圖標順序

您可以使用order樣式分配了一個數字。例如， 5將在進度條之後但在音量按鈕之前放置一個圖標，並且9會將圖標放置在控制欄的最右側。對於較舊的瀏覽器，可能需要更多樣式，如下所示：

{
  order: 5; /* after the progress bar but before the volume button */
  -webkit-box-ordinal-group: 5;
  -moz-box-ordinal-group: 5;
  -webkit-order: 5;
  -ms-flex-order: 5;
}

添加圖標的示例（在這種情況下為下載文件）顯示在Brightcove Player示例：下載視頻插件文檔。

CSS特異性

CSS特殊性是指瀏覽器根據相關性確定將哪些屬性值應用於元素的方式。有關此概念的詳細信息，請查看專一性文章。

在Chrome中，檢查大播放按鈕。請注意background屬性是使用多個類設置的。這意味著我們不能只使用一個.vjs-big-play-button類以覆蓋此屬性。

以下是自定義大播放按鈕的三種方法：

  • 引用分配給播放器的ID。
  • 使用多個類來提高屬性的相關性。
  • 使用JavaScript在播放器上設置id屬性，然後在CSS中引用它。

引用一個ID

由於一個id屬性與CSS特異性相關性很高，因此添加了一個id CSS選擇器將覆蓋background-color財產。

如果您尚未這樣做，請添加id歸因於video元素如下：

    <video-js id="video_1"
      data-account="1752604059001"
      data-player="68f30408-8fb4-431d-accb-8c5baa8c4790"
      data-embed="default"
      class="video-js" controls></video-js>

在CSS中，將id值添加到.vjs-big-play-button類選擇器。覆蓋背景顏色和不透明度，如下所示：

  <style>
    #video_1 .vjs-big-play-button {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

您應該看到大播放按鈕現在變為橙色。

使用多個類

自從background大播放按鈕的屬性是使用兩個類設置的，因此在CSS選擇器中至少需要兩個類才能覆蓋此屬性。

添加.video-js上課.vjs-big-play-button類選擇器。請記住.video-js類在視頻容器上設置。

使用這些值，就是說選擇所有類名稱為的元素vjs-big-play-button是類名稱為的元素的後繼者.video-js。

<style>
  .video-js .vjs-big-play-button {
    background-color: #B37D5B;
    opacity: .6;
  }
  .video-js {
    width: 640px;
    height: 360px;
  }
</style>

您應該看到大播放按鈕現在變為橙色。使用多個類可能很棘手，並且可能會影響多個元素，而id特定於一個元素。

使用JavaScript

您還可以使用JavaScript向大播放按鈕動態添加ID。然後，您可以從CSS中引用該ID。

添加代碼以向大播放按鈕元素添加ID。

  • 173行：獲取對具有以下類的第一個元素的引用vjs-big-play-button。
  • 第174行：添加一個id歸因於大播放按鈕。
  <script type="text/JavaScript">

    var playButton = document.getElementsByClassName("vjs-big-play-button")[0];
    playButton.setAttribute("id", "myPlayButton");

  </script>

在CSS中，更改選擇器以使用myPlayButton ID名稱。

  <style>
    #myPlayButton {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

您應該看到大播放按鈕現在變為橙色。
