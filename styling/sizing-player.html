調整播放器大小

本主題向您展示如何使用iframe和embed_in_page實現調整視頻和播放器的大小。

尺碼考量

調整視頻和播放器大小時，要記住兩個重要概念：

  • 長寬比–在創建播放器時在curl語句中設置的媒體寬度和高度屬性定義了視頻的固有尺寸，也稱為長寬比。

  • 播放器大小–在CSS或HTML代碼中設置的播放器寬度和高度屬性控製播放器的大小。更改播放器的大小時，它將使用視頻固有尺寸來保持控件正確放置。

設定長寬比

視頻寬高比是寬度和高度之間的關係。經典電視使用4：3的縱橫比，而寬屏使用16：9的縱橫比。

創建播放器時，可以使用Studio指定視頻的寬度和高度。使用玩家們模塊，玩家信息部分。

注意：要確定視頻的長寬比，只需將高度除以寬度即可。例如，高度為 360 像素，寬度為 640 像素的視頻的比例為 .5625 (360/640 = .5625)，即 16:9 (9/16 = .5625)。

視頻寬高比將保持不變，並且隨著播放器做出相應的調整，播放器控件將被正確放置。

使用錯誤的比率

如果您指定的媒體寬高比與視頻的尺寸不同，則會在視頻周圍出現文字框，並且播放器控件將不會位於視頻上方。

在這裡，我們將媒體寬度和高度分別設置為640px和480px（長寬比為4：3），同時使用長寬比為16：9的Tiger視頻。

使用正確的比例

當媒體尺寸與視頻的長寬比匹配時，則沒有信箱裝箱，並且在調整播放器大小時，播放器控件將保持正確的位置。

在這裡，我們將媒體的寬度和高度分別設置為640px和360px（長寬比為16：9），與Tiger視頻相同。

設定播放器大小

您可以使用Studio設置播放器的大小。使用玩家們模塊，玩家信息部分。

您還可以使用CSS設置播放器的大小，如以下各節所述。

使用iframe

對於iframe實施，可以通過提供width和height的屬性iframe元素。做這件事有很多種方法：

  • 將屬性添加到iframe元素

  • 將CSS與iframe元素

  • 將CSS與id屬性

注意：通過/樣式/response-sizing-brightcove-player.html設置width和height屬性，您可以創建大小固定的播放器。若要建立回應式調整大小的播放器，請參閱 Brightcove 播放器的回應式尺寸調整文件。

將屬性添加到iframe元素

添加並設置width和height的屬性iframe元素。

<!doctype html>
<html>

<head>
  <meta charset="UTF-8">
  <title>Size the Player</title>
</head>

<body>
  <iframe src='https://players.brightcove.net/1752604059001/0711ceb1-07cf-430c-bd6f-951937e2c488_default/index.html'
    allowfullscreen webkitallowfullscreen mozallowfullscreen
    width="640" height="360"></iframe>

</body>
</html>
當使用width和/或height與 iframe 或頁內嵌入播放器實現內聯的屬性，單位為像素。您可以包括px數字之後，但從技術上講，這不是正確的HTML語法。見使用百分比本文檔後面的部分介紹了使用百分比值的方法。

將CSS與iframe元素

在CSS中，將width和height的屬性iframe元素。

<!doctype html>
<html>

<head>
  <meta charset="UTF-8">
  <title>Size the Player</title>

  <!-- Page styles -->
  <style>
    iframe {
      width: 640px;
      height: 360px;
    }
  </style>
</head>

<body>
  <iframe src='https://players.brightcove.net/1752604059001/0711ceb1-07cf-430c-bd6f-951937e2c488_default/index.html'
    allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>

</body>
</html>

將CSS與id屬性

在CSS中，請參考id添加到iframe元素。如果您有多個，則此方法很有用iframe頁面上的元素。

<!doctype html>
<html>

<head>
  <meta charset="UTF-8">
  <title>Size the Player</title>

  <!-- Page styles -->
  <style>
    #myPlayer {
      width: 640px;
      height: 360px;
    }
  </style>
</head>

<body>
  <iframe id="myPlayer"
    src='https://players.brightcove.net/1752604059001/0711ceb1-07cf-430c-bd6f-951937e2c488_default/index.html'
    allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>

</body>
</html>

使用embed_in_page

對於embed_in_page實現，可以通過提供width的高度和高度屬性video元素。做這件事有很多種方法：

  • 將屬性添加到video元素
  • 將CSS與class屬性一起使用
  • 將CSS與id屬性
注意：通過設置width和height屬性，您可以創建大小固定的播放器。若要建立回應式調整大小的播放器，請參閱 Brightcove 播放器的回應式尺寸調整文件。

將屬性添加到video元素

添加並設置width和height的屬性video元素。

<!doctype html>
<html>

<head>
  <meta charset="UTF-8">
  <title>Size the Player</title>
</head>

<body>

  <video-js
    data-account="1752604059001"
    data-player="860a1ffd-4be3-4df6-9d46-c0621534108b"
    data-embed="default"
    class="video-js" controls
    width="640" height="360"></video-js>

  <script src="https://players.brightcove.net/1752604059001/860a1ffd-4be3-4df6-9d46-c0621534108b_default/index.min.js"></script>

</body>
</html>

將CSS與class屬性一起使用

這video元素具有一個class屬性，可以從CSS引用該屬性來設置播放器大小。

<!doctype html>
<html>

<head>
  <meta charset="UTF-8">
  <title>Size the Player</title>

  <!-- Page styles -->
  <style>
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>
</head>

<body>
  <video-js
    data-account="1752604059001"
    data-player="0711ceb1-07cf-430c-bd6f-951937e2c488"
    data-embed="default"
    class="video-js" controls></video-js>

  <script src="https://players.brightcove.net/1752604059001/0711ceb1-07cf-430c-bd6f-951937e2c488_default/index.min.js"></script>

</body>
</html>

使用百分比

使用時height和/或width Brightcove Player的屬性iframe或者video標籤，這些屬性以像素為單位只要。實際上，對於height和width屬性（您可以輸入px，但從技術上講，它是無效的HTML）：

width="480"

您可以使用百分比，並且可以通過兩種不同的方式使用CSS來實現：

  • 用一個style中的屬性video標籤。 
    <video-js data-video-id="5622718636001"
  data-account="1507807800001"
  data-player="default"
  data-embed="default"
  data-application-id
  class="video-js"
  controls
  style="width: 100%;height: 100%"></video-js>
<script src="https://players.brightcove.net/1507807800001/default_default/index.min.js"></script>
  • 用一個style標籤。 
    ...
<style>
    .video-js {
      height: 100%;
      width: 100%;
    }
</style>
...
<video-js data-account="1752604059001"
  data-player="H1xW7NWcz"
  data-embed="default"
  data-application-id
  class="video-js"
  controls></video-js>
<script src="https://players.brightcove.net/1752604059001/H1xW7NWcz_default/index.min.js"></script>
如果您單獨使用百分比，並且玩家不在具有高度設置的容器中，則可能您沒有看到播放器，因為高度為零。確保播放器的父容器有一個高度設置。

全屏顯示

在支持全屏API的較新瀏覽器中，有必要應用頁內CSS規則，以確保在切換到全屏時將播放器縮放到100％。否則，播放器將以全屏顯示的原始尺寸顯示。

使用以下CSS確保播放器在全屏模式下可縮放：

/***************************
* Fullscreen display      *
***************************/
body.vjs-full-window {
  padding: 0;
  margin: 0;
  height: 100%;
}
.video-js.vjs-fullscreen {
  position: fixed;
  overflow: hidden;
  z-index: 1000;
  left: 0;
  top: 0;
  bottom: 0;
  right: 0;
  width: 100% !important;
  height: 100% !important;
}
.video-js:-webkit-full-screen {
  width: 100% !important;
  height: 100% !important;
}
.video-js.vjs-fullscreen.vjs-user-inactive {
  cursor: none;
}

優先順序

您已經看到了多種播放器尺寸的方法。以下是順序優先級的列表：

  • 運行時更改（使用JavaScript更改播放器大小）
  • 在<video-js>標記中設置的大小
  • CSS（遵循正常的繼承規則）
  • 播放器配置設置

Player API中的大小

Player API中有4種播放器方法，它們返回以像素為單位的大小，但是了解它們之間的區別很重要。

  • .height() - returns the height set in the height attribute of the video tag (if that attribute is not set, height() will return 0)
  • .width() - returns the width set in the width attribute of the video tag (if that attribute is not set, width() will return 0)
  • .currentHeight() - returns the current height of the player, however it was set
  • .currentWidth() - returns the current width of the player, however it was set

在大多數情況下，currentHeight()和currentWidth()是您要使用的方法。
