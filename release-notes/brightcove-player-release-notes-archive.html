    布萊特灣玩家發行紀錄

    本頁追蹤布萊特灣玩家發行資訊

    注意：根據預設，更新會自動套用至所有玩家。若要防止玩家自動更新，您必須將其設定為手動更新 (也稱為鎖定播放器)。您可以在 Studio 播放器模塊（外部網站）或使用播放器管理 API 中執行此操作。

    選擇退出自動更新應該是暫時性的量值，當您解決更新所建立的任何問題時套用。如果您沒有將玩家更新到當前版本，您錯過了新功能和錯誤修復，而舊版本的播放器將在某個時候停止支持

    也，您可以在以下位置查看Brightcove Player版本和插件版本Brightcove Player發行公告 Google網上論壇。

    版本6-v6.45.4

    2020年10月9日

    • 更新到VHS 2.1.0
    • 更新至Video.js 7.9.3
    • 修正影片內嵌language屬性支援的回歸問題。該用例已被棄用；language不是<video>或<video-js>元素的有效屬性！
    • 修復了預覽和帶有查詢參數的文件URL失敗的問題
    • 現在在播放器信息模式中顯示DRM版本（如果可以找到）
    • VHS 2-包含VHS 2.1.0 Brightcove Player中的內容對於Brightcove是一個巨大的里程碑。播放引擎的重大更新為我們提供了許多未來的增強功能。它代表了經過數月的幕後努力，測試和完善。從觀看者的角度來看，玩家行為不會有差異。祝賀所有為交付VHS 2做出貢獻的人！
    • 設置policy_key至沒有在播放器配置中將不再起作用（某些情況下會這樣做）。相反，設置policy_key至零

    版本5-5.28.1

    不再支持Brightcove Player版本5。已發布的播放器將繼續工作並且不會被禁用，但是Brightcove無法保證將來與瀏覽器，API，視頻格式，集成以及播放器外部可能發生的其他更改兼容。

    我們的建議是升級到版本6的最新版本。如果您需要任何幫助，請聯繫Brightcove支持。

    新的功能

    Brightcove Player版本5在不使用Flash的情況下對HLS視頻提供了新的支持。在此版本中，在所有支持Media Source Extensions API（MSE）的瀏覽器中，Brightcove Player都可以播放HLS視頻沒有閃光燈！這意味著在Firefox，Chrome，Safari或Microsoft Edge上不使用Flash。Flash仍將用於較舊版本的IE，但可以通過配置禁用。

    MSE HLS具有幾個巨大的好處：

    1. 即使在高分辨率和高幀頻下，視頻也可以快速加載並流暢播放，包括4K和60FPS視頻-在適度的硬件上
    2. 它大約使用-50％的CPU負載來渲染相同的視頻，這意味著電池可以使用更長的時間，並且筆記本電腦的風扇無法打開
    3. 省電和其他瀏覽器設置以限制Flash使用不再對玩家有任何影響不論玩家的身高

    此版本中的其他功能包括：

    1. 新皮膚 -根據重要的UX研究，我們構建了一個包含許多更改的新外觀。其中包括一個新的居中大播放按鈕，新顏色，新搜索欄行為以及一個新的標題欄（稱為停靠欄）。可用性測試參與者特別喜歡新的時間表，該時間表會隨著懸停而擴展，並使其更容易找到特定時間。現在，您可以在默認的新皮膚（稱為Luna）和舊的1.x皮膚（現在稱為Graphite）之間進行選擇。
    2. 初步支持實時字幕 -此版本現在支持許多608/708字幕編碼系統。這包括對流行的彈出和匯總方法以及基本樣式的支持
    3. 兼容模式適用於所有1.x和Video.js 4.x插件。 （DASH / DRM，FreeWheel和OnceUX插件除外-在第一季度將進行更新）

    同樣在此版本中，我們進行了許多更新以改進用於定制的編程模型，包括簡化了外觀模型和插件API。這些更改為我們帶來了最新的開源版本Video.js 5。您還將注意到，我們還在Video.js和Brightcove Player版本之間同步了版本編號。
