Note that every 1-2 weeks pre-released versions become the current version.那時所有非鎖定播放器都升級到了當前版本。如果您出於某種原因希望將播放器鎖定在某個版本上，請參閱選擇退出自動播放器更新（外部站點） .
 
 

試用新版本

如果有預發布版本，則試用預發布版本的最簡單方法是轉到Studio的選手們模塊，單擊要更改的播放器，然後編輯玩家信息。使用版本選擇器選擇最新的預發布版本：

您可以使用curl語句將播放器更新為預發行版本。捲曲的快速概述可以在頂部找到一步步：播放器管理（外部站點）。在文檔中有一個特定的部分，關於如何更新您的播放器。

這是curl命令，用於更新播放器，在其中替換為的適當值你的郵件， YOUR_ACCOUNT_ID和YOUR_PLAYER_ID。 

curl \
  --header "Content-Type: application/json" \
  --user YOUR_EMAIL \
  --request PATCH \
  --data '{
    "player": {
    "template": {
      "version": "6.57.0"
      }
    }
  }' \
  https://players.api.brightcove.com/v2/accounts/YOUR_ACCOUNT_ID/players/YOUR_PLAYER_ID/configuration

在大多數情況下，您將需要發布播放器，這可以通過以下curl語句來完成。同樣，您將需要在指示的地方替換正確的值。 

curl \
  --header "Content-Type: application/json" \
  --user YOUR_EMAIL  \
  --request POST \
  https://players.api.brightcove.com/v2/accounts/YOUR_ACCOUNT/players/YOUR_PLAYER_ID/publish
