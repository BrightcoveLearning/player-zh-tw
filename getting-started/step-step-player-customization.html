一步步：玩家定制

概述

本文檔將分為幾節，內容介紹如何將CSS與頁面內嵌入一起使用，並且播放器的iframe實現是完全不同的。

本文檔中的內容使用Chrome的開發工具，但所有現代瀏覽器都具有等效的功能。有關在不同瀏覽器中使用開發工具的簡要介紹，請參見調試基礎文檔。

注意：Video Cloud Studio內部提供了一些基本的播放器樣式。如需詳細資訊，請參閱造型播放器。

確定玩家選擇器

CSS選擇器使您可以選擇和操作頁面上的HTML元素。CSS選擇器用於根據其ID，類，類型，屬性等來“查找"（或選擇）HTML元素。嘗試更改HTML頁面上元素的外觀時，您必須做的第一件事就是確定用於尋址元素的選擇器。以下步驟顯示瞭如何找到播放器的選擇器，然後更改播放器的大小。

  1. 在 Video Cloud Studio 中，打開媒體模塊並選擇要發布的視頻。
  2. 選擇您要與之一起播放視頻的播放器。
  3. 設置漿紗，長寬比和播放器大小。
  4. 複製先進的發布代碼。
  5. 使用您的HTML編輯器，創建一個新的HTML頁面。
  6. 將發布代碼粘貼到HTML頁面的正文中。
  7. 瀏覽新創建的頁面。
  8. 打開瀏覽器的開發工具檢查 HTML 代碼並深入到 HTML 中找到<video-js>標籤。您會看到<video-js>標記具有一個也命名為的類video-js。這<video-js>標籤將始終具有video-js。您將利用video-js設置播放器樣式時經常上課。
  9. 查看下面的屏幕截圖，以紅色突出顯示該課程。綠色突出顯示的是<video-js>標籤，當然是Brightcove Player。

播放按鈕選擇器

將使用另一種方法來找到播放按鈕的選擇器。以下步驟顯示瞭如何為播放按鈕找到正確的選擇器，然後更改其外觀。

  1. 瀏覽您的HTML頁面。
  2. 右鍵單擊覆蓋播放器的播放按鈕（您必須右鍵單擊兩次，因為第一次右鍵單擊會顯示 Brightcove 播放器信息）並選擇檢查 .
  3. 請注意<div>突出顯示的標記表示播放按鈕，並具有vjs-big-play-button。
  4. 更改樣式以包括CSS，該CSS會更改播放按鈕的顏色和不透明度。 
    <style>
  .vjs-big-play-button {
    background-color: red;
    opacity: .3;
  }
</style>
  5. 瀏覽頁面，您將看到並非所有樣式都已使用。該按鈕不是紅色。
  6. 在開發人員工具的右側查看，您將看到樣式的摘要。您可以向下看，將背景更改為紅色被覆蓋了，以紅色表示出來。
    注意：在 Safari 中，即使樣式被覆蓋，也可能不會被刪除。

    發生的事情是對背景顏色進行了許多更改。您設置的樣式已在CSS的“層疊"部分中被覆蓋。您可以通過利用稱為CSS的功能來更改此設置特異性。特異性確定瀏覽器將應用哪個CSS規則，並且是一種權重類型，它與級聯樣式表（CSS）規則的顯示方式有關。使用特定性，可以使樣式比其他樣式更重要，從而強制使用它。有關CSS專用性的簡要介紹，請參見自定義播放器外觀 doc，以及任何搜索中的大量參考文獻。

  7. 在這種情況下，使用video-js類將“增強"您的樣式的優先權。更改樣式以使用新的組合選擇器。請注意，最後一堂課之前有一個空格（為什麼可以在特異性規則中找到其詳細信息）。 
    <style>
  .video-js .vjs-big-play-button {
    background-color: #ff0000;
    opacity: .3;
  }
</style>
  8. 瀏覽您的頁面，看到所有樣式現在都已應用。
  9. 再次查看開發人員工具的右側，您將看到樣式的摘要。現在，您將看到背景顏色更改沒有被覆蓋。

高級代碼實施

當使用上述選擇器和样式時，結果將與播放器的高級代碼實現一起使用，如下所示：

<!doctype html>
<html>

<head>
  <meta charset="UTF-8">
  <title>Untitled Document</title>
  <style>
    .video-js .vjs-big-play-button {
      background-color: red;
      opacity: .3;
    }
  </style>
</head>

<body>

  <video-js data-account="1507807800001"
    data-player="default"
    data-embed="default"
    controls=""
    data-video-id="6065136830001"
    data-playlist-id=""
    data-application-id=""
    width="640" height="360">

  </video-js><script src="https://players.brightcove.net/1507807800001/default_default/index.min.js"></script>

</body>

</html>

這是在HTML頁面上使用CSS的一種非常常見的方式。你看<style>區塊在頁面上<video-js>標籤。

注意：根據您的CSS經驗，您可能想知道為什麼上面的代碼沒有分配一個id歸因於<video-js>標籤。然後，您可以使用類似的選擇器#myPlayer .vjs-big-play-button。實際上，您可以擁有並且它會起作用。問題是，使用iframe播放器實現時，使用id選擇器創建的CSS無法使用，因此僅使用了現有的類名。通過不使用id屬性，這些樣式將直接適用於iframe播放器實現。

標準代碼實施

毫無疑問，標准或iframe代碼實現將Brightcove Player放置在iframe中，然後將其放置在HTML頁面中。您可以使用包含iframe的頁面上的樣式來設置實際iframe本身的樣式，例如，更改其大小，但是iframe頁面上的樣式請勿層疊到iframe本身。iframe確實會拉入單獨的HTML頁面，因此，iframe內容的所需樣式必須與iframe中的HTML頁面相關聯。

使用Brightcove Player的iframe實現時，應用CSS的步驟為：

  • 將樣式放置在Internet上可訪問的文件中
  • 將樣式錶鍊接到播放器（使用Studio或播放器管理API完成）

注意：以下步驟假定您將使用上一步中完成的HTML進行工作，並顯示在上面的代碼片段中。如果您希望保留該示例作為工作示例，只需將代碼複製到另一個文件中，然後在新文件中開始以下步驟。

  1. 在Studio中，發布視頻並複制代碼，以實現播放器的iframe實施。
  2. 在HTML頁面中，將頁面內嵌入代碼替換為iframe代碼。
  3. 檢查以確保您的HTML頁面看起來類似於以下內容： 
    <!doctype html>
<html>

<head>
  <meta charset="UTF-8">
  <title>Untitled Document</title>
  <style>
    .video-js .vjs-big-play-button {
      background-color: red;
      opacity: .3;
    }
  </style>
</head>

<body>

<iframe src="https://players.brightcove.net/1507807800001/default_default/index.html?videoId=6065136830001"
  allowfullscreen=""
  allow="encrypted-media"
  width="640" height="360"></iframe>

</body>

</html>
  4. 瀏覽頁面，您將看到未使用按鈕的樣式。
  5. 創建一個名為custom-qs.css並將其保存在Internet上可訪問的位置。
  6. 剪切按鈕的樣式並將其粘貼到文件中並保存。僅剪切按鈕的樣式，不能包含按鈕的樣式<style>標籤。該文件應僅包含以下內容： 
    .video-js .vjs-big-play-button {
  background-color: red;
  opacity: .3;
}
  7. 檢查以確保您的HTML頁面看起來類似於以下內容： 
    <!doctype html>
<html>

<head>
  <meta charset="UTF-8">
  <title>Untitled Document</title>
</head>

<body>

<iframe src="https://players.brightcove.net/1507807800001/default_default/index.html?videoId=6065136830001"
  allowfullscreen=""
  allow="encrypted-media"
  width="640" height="360"></iframe>

</body>

</html>
  8. 在Studio中，進入玩家們模塊並編輯用於此快速入門的播放器。
  9. 點擊插件在左側導航菜單中。
  10. 點擊樣式表，然後添加樣式表按鈕，然後輸入您的路徑：
  11. 按一下 [ 儲存] 儲存變更。頁面右側的預覽播放器將更新，以顯示應用於按鈕的CSS樣式。
  12. 若要發佈播放程式，請按一下「發佈與內嵌」>「發佈變更」。
  13. 若要關閉開啟對話方塊，請按一下「關閉」。
  14. 返回瀏覽器並刷新頁面。啟用發布的更改後，您將看到您的樣式已應用於iframe播放器。可能要過幾分鐘才能看到發布的更改。
