一步步：Brightcove播放器

{{ page.description }}

概述

本文檔提供了動手入門，介紹瞭如何創建Brightcove Player，然後使用兩種不同的代碼實現來使用新創建的播放器來發布視頻。您還將以編程方式開始播放視頻，然後將一個插件添加到播放器。

在此快速入門中，您將執行以下任務：

  • 使用Studio創建播放器
  • 使用Studio在播放器中發布視頻
  • 使用iframe播放器實施
  • 使用頁內嵌入實施
  • 以編程方式播放視頻
  • 將疊加插件添加到播放器

創建播放器

若要創建一個新的播放器，請按照下列步驟。

  1. 登錄到Studio，網址為https://studio.brightcove.com/。
  2. 當前帳戶名稱將顯示在頁面的右上角。如果您有多個帳戶，請點擊經常賬戶鏈接並選擇您要在其中創建播放器的帳戶。
  3. 點擊玩家們導航標題中的鏈接。
  4. 點擊新玩家按鈕。
  5. 進入一個名稱和簡短的介紹為玩家。為了快速入門，播放器名稱將為樣本播放器。
  6. 點擊保存。
  7. 確認新創建的播放器出現在播放器列表中。

在播放器中發布視頻

您可以在“媒體"模塊中新創建的播放器中發布視頻。將視頻與播放器關聯後，您便可以發布視頻，然後可以訪問三種實現方式，以便在播放器中查看視頻。

要在播放器中發布視頻，請按照以下步驟操作：

  1. 在Studio中，通過點擊媒體主菜單中的鏈接。
  2. 單擊您希望在播放器中發布的沒有音頻軌道的視頻所在的行，然後單擊發布和嵌入>發佈到網絡 .
    筆記：如果視頻有音軌，則它不會自動播放，您希望在文檔的稍後部分進行播放。查看文檔自動播放注意事項有關無需用戶交互即可播放視頻的更多信息。
  3. 來自選擇一個球員下拉菜單，選擇您新創建的播放器。
  4. 現在，您可以訪問播放器的三個實現：標準（iframe），先進的（頁內）和一個在瀏覽器中預覽網址。不要關閉此窗口，因為在此快速入門中，您將無數次從中復制代碼。
Note:如果好奇的話，您可以使用URL代碼查看播放器配置的“內幕"。點擊在瀏覽器中預覽 URL鏈接。在瀏覽器中，更改index.html URL的末尾config.json。您將看到一個 JSON 對象，詳細說明您剛剛創建的播放器的當前配置。

您將使用JSON編輯器將視頻添加到播放器。要配置視頻內容，請按照下列步驟操作。

  1. 點擊鏈接樣本播放器打開播放器屬性頁面。
  2. 點擊JSON編輯器在左側導航菜單中。
  3. 在JSON的第二個到最後一個花括號後放置一個逗號。
  4. 在最後一個大括號上方，添加以下內容以將視頻和海報分配給播放器。當然，您可以替換自己的視頻和海報。

 "src": "//solutions.brightcove.com/bcls/assets/videos/BirdsOfAFeather.mp4", "type": "video/mp4"
 }],
 "poster": {
 "highres": "//solutions.brightcove.com/bcls/assets/images/BirdsOfAFeather.jpg"
 }
 }

Publishing the player will push the changes to the published player.有關預覽播放器，已發布的播放器和播放器發布代碼的更多信息，請參見生成播放器嵌入 代碼。要發布播放器，請按照下列步驟操作
  1. 點擊發布並嵌入然後發布變更發布您的播放器。
  2. 在發布更改的同一對話框中，您還可以復制標準（iframe）或高級（頁面內嵌入）播放器實施代碼。請注意，實現代碼將反映漿紗 ( 自適應 或者 固定的 ）， 長寬比 和 方面 所做的選擇並保存在玩家信息選項。
  3. 將兩種代碼實現類型都複製到文本文件中，以備後用。

使用標準代碼

Note:對於標準代碼（iframe實現），由於iframe的要求，您將需要使用Web服務器瀏覽頁面。僅在本地查看頁面是行不通的。
  1. 創建一個HTML頁面並粘貼標準（iframe）代碼從上一步插入到正文中。
  2. 當你瀏覽這個頁面時，你會看到視頻反映尺寸（ 自適應 或者 固定的 )、「外觀比例」和「尺寸」選項。

使用高級嵌入 代碼

在本節中，您將使用高級（頁內嵌入）代碼實現。

  1. 返回 Video Cloud Studio 中的發布對話框。
  2. 觀看先進的代碼並單擊固定的單選按鈕漿紗 .
    這反應靈敏 sizing 包括 HTML 以使播放器響應。在此活動中，重點是播放器代碼，因此將使用更簡單的固定的漿紗。
  3. 創建一個HTML頁面並粘貼先進的（頁內嵌入）代碼從上方進入正文。
  4. 檢查代碼，該代碼看起來將類似於以下內容，並記下屬性： 
     <video-js data-account="1507807800001"
  data-player="EUYJo0AOB"
  data-embed="default"
  controls=""
  data-video-id="6071787405001"
  data-playlist-id=""
  data-application-id=""
  width="640" height="360"></video-js>
<script src="https://players.brightcove.net/1507807800001/default_default/index.min.js"></script>
    • data-account：帳號
    • data-player：Brightcove播放器的ID
    • data-embed：任何一個default如果播放器是父播放器，或者是父播放器的ID；看嵌入式API指南有關嵌入的更多信息（父母/兒童玩家）
    • controls：如果播放器控件應存在，則用作布爾值（controls標籤中包含的屬性）或未顯示（controls標籤中沒有屬性）
    • data-video-id :Video Cloud 視頻的 ID（如果提供播放器 ID，則不要使用）
    • data-playlist-id :Video Cloud 播放列表的 ID（如果提供視頻 ID，則不要使用）
    • data-application-id :允許您重複使用單個播放器，但可以根據每個站點或每個應用程序區分分析；看將應用程序ID添加到播放器中 代碼想要查詢更多的信息
    • width & height :設置播放器尺寸。
    • <script> tag:每當您使用頁內代碼時，都會附帶script標籤;引用的JavaScript文件包含與特定播放器關聯的所有JavaScript和CSS
  5. 檢查代碼，該代碼看起來將類似於以下內容，並記下屬性：

 data-application-id

 controls></video>
 <script src="https://players.brightcove.net/1507807800001/H15p1gTkg_default/index.min.js"></script>
    • data-account :帳號
    • data-player :Brightcove播放器的ID
    • data-embed :任何一個default如果播放器是父播放器，或者是父播放器的ID；看嵌入式API指南有關嵌入的更多信息（父母/兒童玩家）
    • class :與頁內嵌入代碼關聯的標準Brightcove Player CSS類
    • controls :如果播放器控件應存在，則用作布爾值（controls標籤中包含的屬性）或未顯示（controls標籤中沒有屬性）
    • <script> tag :與特定播放器關聯的所有關聯的JavaScript和CSS
  6. 當你瀏覽這個頁面時，你會看到視頻反映尺寸（ 固定的 )、「外觀比例」和「尺寸」選項。

以編程方式播放視頻

Brightcove Player具有豐富的API。在本節中，您將使用play()以編程方式開始播放視頻的方法。

  1. 添加一個id歸因於video值為的標籤myPlayerID。 
     <video id="myPlayerID"
  2. 在收盤價之上body標記，插入一個script堵塞。 
     <script> </script>
 </body>
  3. 在腳本塊中，使用on()聽的方法loadedmetadata事件。調度事件後，將調用匿名事件處理程序函數。 
     videojs.getPlayer('myPlayerID').on('loadedmetadata', function() { });
    Note:有關 Brightcove Player 啟動事件的更多信息，請參閱啟動事件的部分了解玩家事件文檔。
  4. 創建函數局部變量，名為myPlayer，它指定播放器實例，引用為this，到該變量。 
     var myPlayer = this;
  5. 由於大多數瀏覽器不再允許自動播放帶有音軌的視頻，因此最好使用muted()靜音視頻並確保視頻將自動播放的方法。 
     myPlayer.muted(true);

    見自動播放注意事項文檔以獲取有關自動播放問題的更多信息。

  6. 使用播放器的play()播放視頻的方法。 
     myPlayer.play();
  7. 確認您的腳本塊顯示如下。 
     <script> 
   videojs.getPlayer('myPlayerID').on('loadedmetadata', function() { 
     var myPlayer = this;
     myPlayer.muted(true);
     myPlayer.play();
   });
 </script>
  8. 保存更改並在瀏覽器中刷新頁面，以觀看視頻自動開始播放。如果選擇了包含音軌的視頻，則必須單擊“播放"按鈕才能播放視頻。

添加插件

Brightcove Player具有許多可用來增強播放器的插件。一種這樣的插件是覆蓋插件。在本節中，您將將該插件添加到您在上一節中完成的高級代碼中。有關插件的詳細信息，請參見顯示覆蓋插件文檔。

使用插件時，必須提供作為插件實現的JavaScript的路徑。如果插件需要，您可能還必須提供指向CSS的鏈接。

  1. 在收盤價之上head標籤，插入以下鏈接語句，該語句指向覆蓋插件的CSS位置。 
     <link href="https://players.brightcove.net/videojs-overlay/lib/videojs-overlay.css" rel='stylesheet'> </head>
  2. 在您之前添加的腳本塊上方，添加以下內容script標記，它是實現覆蓋插件功能的實際JavaScript代碼。 
     <script src="https://players.brightcove.net/videojs-overlay/lib/videojs-overlay.js"></script> <script>
 videojs.getPlayer('myPlayerID').on('loadedmetadata', function() {
  3. 就在哪裡play()使用方法時，調用播放器的overlay()方法，並添加括號作為參數，以準備傳遞JSON配置對象。 
     myPlayer.overlay({ });
  4. 可以將多個覆蓋對像作為數組傳遞給插件，以在不同時間和不同位置顯示不同的內容。在這種情況下，將僅傳遞一個疊加層對像以顯示一些文本，並且只有在播放視頻時才顯示該文本。視頻暫停時，將不會顯示疊加層。要實現此目的，請添加以下疊加層對像作為配置參數。 
     overlays: [{ 
  content: 'This event-triggered overlay message appears when the video is playing', 
  start: 'play',
  end: 'pause'
}]
  5. 確認您的腳本塊顯示如下，並且是否已正確調用並配置了疊加插件。 
     <script> 
  videojs.getPlayer('myPlayerID').on('loadedmetadata', function() {
    var myPlayer = this;
    myPlayer.muted(true);
    myPlayer.play();
    myPlayer.overlay({
      overlays: [{
        content: 'This event-triggered overlay message appears when the video is playing', 
        start: 'play',
        end: 'pause'
      }]
    });
  });
</script>
  6. 保存更改並在瀏覽器中刷新頁面，以觀看視頻自動開始播放。視頻開始播放時，您會看到疊加層出現。暫停視頻以查看已刪除的疊加層。
    Note:此處的“覆蓋插件"展示的實現僅與特定HTML頁面上的播放器一起使用。您可以將插件添加為播放器的一部分，而不管播放器在何處使用了多少次，而無需重複複製和粘貼代碼。見一步步：插件開發查看面向開發人員的實現方法，或配置播放器插件文檔以了解如何使用 Studio 在播放器中實現插件。
