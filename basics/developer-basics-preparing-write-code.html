玩家開發者基本知識：準備編寫代碼

在本主題中，您將學習如何創建標準的最佳實踐模板，以準備編寫代碼來控製播放器。

腳步

  1. 將Advanced Implementation Player代碼的實例粘貼到HTML頁面中。 
    <video data-video-id="5831704295001"
  data-account="921483702001"
  data-player="Uj7Yz80yM"
  data-embed="default"
  data-application-id
  class="video-js"
  controls
  width="640"
  height="360"></video>
<script src="//players.brightcove.net/921483702001/Uj7Yz80yM_default/index.min.js"></script>
  2. 添加一個id歸因於<video>標記並為其分配值myPlayerID。 
    <video id="myPlayerID"
  3. 添加一個<script>剛好在</body>標籤。
  4. 在裡面script塊，使用videojs.getPlayer()獲取頁面上播放器引用的方法。使用id之前添加為方法的參數。 
    <script>
  videojs.getPlayer('myPlayerID')
</script>
  5. 使用ready()方法，並編寫一個匿名回調函數作為其參數。 
    <script>
  videojs.getPlayer('myPlayerID').ready(function() {

  )};
</script>
  6. 聲明一個變量，名為myPlayer在匿名函數中並為其分配關鍵字this，它代表匿名功能上下文中的播放器。 
    var myPlayer = this;

完整的代碼

<!doctype html>
<html>

<head>
  <meta charset="UTF-8">
  <title>Preparing to Write Code</title>
</head>

<body>

<video id="myPlayerID"
  data-video-id="5831704295001"
  data-account="921483702001"
  data-player="Uj7Yz80yM"
  data-embed="default"
  data-application-id
  class="video-js"
  controls
  width="640"
  height="360"></video>
<script src="//players.brightcove.net/921483702001/Uj7Yz80yM_default/index.min.js"></script>

<script>
  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;
  )};
</script>

</body>

</html>
