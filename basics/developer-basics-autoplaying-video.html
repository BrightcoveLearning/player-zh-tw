玩家開發者基本知識：先進的播放器代碼

在本主題中，您將學習如何以編程方式自動播放視頻。
 

腳步

  1. 從標準的最佳實踐代碼開始，以編程方式進行控制 玩家。 
    <video id="myPlayerID"
  data-video-id="5785633544001"
  data-account="921483702001"
  data-player="Uj7Yz80yM"
  data-embed="default"
  data-application-id
  class="video-js"
  controls
  width="640"
  height="360"></video>
<script src="//players.brightcove.net/921483702001/Uj7Yz80yM_default/index.min.js"></script>

<script>
  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;

  });
</script>
  2. 在匿名函數中videojs.getPlayer()方法，按照變量分配，將播放器靜音，使其可以自動播放。 
    myPlayer.muted(true);
  3. 使用播放器的on()為該事件添加事件處理函數的方法loadedmetadata事件。 
    myPlayer.on('loadedmetadata', function(){

});
  4. 在事件處理程序中，播放視頻。 
    myPlayer.play()
