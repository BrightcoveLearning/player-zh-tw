玩家開發者基本知識：自定義插件-使用Studio

在本主題中，您將學習如何使用Studio將自定義插件分配給Brightcove Player。
 

腳步

  1. 在Studio中，選擇選手們模塊。
  2. 單擊您要向其添加自定義插件的播放器。
  3. 在播放器屬性中，在左側選擇外掛程式選項 。
  4. 來自添加插件下拉菜單，選擇自定義插件。
  5. 添加以下內容：
    • 插件名稱
    • 網址
    • CSS URL-如果需要
    • 選項（JSON）-如果需要
  6. 保存播放器。
  7. 發佈播放程式。
