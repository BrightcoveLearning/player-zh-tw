驗證VAST廣告代碼

為確保您擁有的VAST廣告代碼在Google上有效，請按照以下步驟操作。

  1. 獲取廣告代碼。（如果沒有，請使用此步驟後的示例進行演示。）在此快速入門中，假設您的廣告包含前貼片廣告，但任何內嵌廣告都可以使用。 
    https://solutions.brightcove.com/bcls/brightcove-player/vmap/simple-vmap.xml
  2. 要驗證廣告代碼的有效性，請確保您已復制其URL並瀏覽到以下頁面：

    視頻套件檢查器（點擊此鏈接將導致該頁面在新窗口或新標籤頁中打開）。如果您使用第1步中提供的網址，則會看到廣告代碼播放前貼片廣告，播放中插片（5秒）和後貼片。

  3. 將廣告代碼網址粘貼到輸入類型表單輸入字段。
  4. 點擊測試廣告您將看到您的廣告播放，然後是Google提供的摩托車比賽視頻。
  5. 確保測試成功完成，然後繼續進行下一部分。如果測試失敗，則需要檢查廣告代碼。
注意：VAST 廣告代碼可以由您的廣告提供商託管，或者您可以復制 XML 並將其託管在您自己的網址上。

啟用廣告

Brightcove Player依靠IMA3插件播放VAST廣告。要將所述插件添加到播放器，請按照以下步驟操作。

  1. 登錄到Studio。
  2. 打開選手們模塊，然後創建新播放器或找到您要向其添加廣告功能的播放器。
  3. 單擊播放器的鏈接以打開播放器的屬性。
  4. 點擊廣告在左側導航菜單中。
  5. 檢查啟用客戶端（IMA）。
  6. 為了廣告代碼，從上方輸入廣告代碼。為了進行測試，您可以使用以下方法： 
    https://solutions.brightcove.com/bcls/brightcove-player/vmap/simple-vmap.xml
  7. 將其他值保留為默認值。你的廣告部分應如下所示：
  8. 點擊保存。
  9. 若要發佈播放程式，請按一下「發佈與內嵌」>「發佈變更」。
  10. 若要關閉開啟對話方塊，請按一下「關閉」。

在播放器中發布視頻

您可以在“媒體"模塊中新創建的播放器中發布視頻。將視頻與播放器關聯後，您便可以發布視頻，然後可以訪問三種實現方式，以便在播放器中查看視頻。

要發布播放器中的視頻，請按照以下步驟操作：

  1. 在Studio中，通過選擇導航到媒體模塊媒體從下拉主菜單中。
  2. 選擇您要發佈到播放器的視頻。在按鈕的第一行中，選擇發布和嵌入然後網絡播放器。
  3. 在選擇控件中，選擇播放器在頁面的“部分"中，選擇您新創建的播放器。
  4. 現在，您可以訪問播放器的三個實現，在瀏覽器中預覽，標準和先進的。不要關閉此窗口，因為您將單擊它的鏈接。
注意：如果好奇的話，您可以使用URL代碼查看播放器配置的“內幕"。點擊在瀏覽器中預覽代碼鏈接，然後復制並瀏覽生成的URL。接下來，更改index.html URL的末尾config.json。您將看到一個 JSON 對象，詳細說明您剛剛創建的播放器的當前配置。

將視頻分配給播放器

視頻內容屬性用於將視頻內容分配給播放器。

要配置視頻內容屬性，請按照下列步驟操作。

  1. 找到影片內容部分，然後單擊編輯。
  2. 添加您選擇的演繹形式。
  3. 添加海報圖像。如果不添加海報圖像，則將使用視頻的第一幀。
  4. 請勿為視頻設置任何尺寸，因為這將通過頁面中的CSS完成。
  5. 點擊保存。
  6. 請按照以下過程發布您的播放器。

測試

當然，您需要確保廣告在播放器中正常運行。

要查看您的播放器中正在播放的音樂，請按照以下步驟操作。

  1. 在這三種實施方式中，點擊在瀏覽器中預覽關聯。
  2. 播放視頻，您將看到您的廣告播放，然後是您分配給播放器的視頻。

當然，您需要確保廣告在播放器中正常運行。在這種情況下，您將使用播放器的預覽版進行測試，因此無需等待播放器實際發布。儘管發佈時間不長，但是使用播放器的預覽版本可以消除由於播放器尚未完成發布而導致無法正常工作的任何混淆。

要查看您的播放器中正在播放的音樂，請按照以下步驟操作。

  1. 在頁面頂部附近，選擇嵌入代碼和網址然後預覽播放器。
  2. 點擊播放器網址。
  3. 播放視頻，您將看到您的廣告播放，然後是您分配給播放器的視頻。
