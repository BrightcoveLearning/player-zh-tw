    使用即時 SSAI 的客戶端預先廣告

    在本主題中，您將了解如何使用 Brightcove 播放器播放用戶端預先廣告，並啟用伺服器端廣告插入 (SSAI) 的即時串流。

    概覽

    使用 Brightcove Player 透過伺服器端廣告插入 (SSAI) 提供即時串流時，您可以插入用戶端的預先廣告。對於客戶端廣告，此功能支援 IMA 廣告。

    此功能需要下列項目：

    • IMA3 外掛程式
    • SAI 外掛程式版本 1.11.0+
    • 布萊特灣玩家版本 6.44.0+

    警告

    • 使用此功能的客戶必須使用動態傳遞。
    • 僅支援 IMA 和 SSAI 外掛程式 (不支援飛輪)。
    • 目前，此功能僅支援網頁上的單一玩家。

    入門

    若要透過即時 SSAI 串流播放用戶端 IMA 預先播放廣告，請依照下列步驟執行：

    1. 建立具有 SSAI 功能的即時串流
    2. 建立布萊特灣玩家
    3. 使用 Studio 實現客戶端預先滾動

    建立具有 SSAI 功能的即時串流

    即時模組支援伺服器端廣告插入 (SSAI)，讓伺服器端廣告可在直播期間要求和顯示。若要建立即時串流，請參閱下列內容：

    • 在即時模組中實作伺服器端廣告

    建立布萊特灣玩家

    創建一個新的布萊特灣播放器使用視頻雲工作室。如需詳細資訊，請參閱快速入門：建立播放程式文件並設定樣式。

    您必須使用布萊特灣播放程式 6.44.0+ 版本。

    使用 Studio 實現客戶端預滾

    若要設定播放器的自動容錯移轉廣告，最簡單的方法就是使用 Video Cloud Studio。一旦您創建了廣告配置和播放器，那麼您就可以準備好配置播放器進行自動容錯移轉，如下所示：

    1. 開啟 PLAYER 模組，找出您要新增廣告功能的播放程式。
    2. 選取播放程式的連結，以開啟播放程式的內容。
    3. 在左側導覽選單中，選取 [ 廣告 ]。
    4. 核取「啟用用戶端 (IMA) 」核取方塊。

    5. 請加入 IMA 廣告代碼的網址。在此範例中，我們將使用範例廣告代碼 URL。

      如需有關播放程式廣告屬性的詳細資訊，請參閱使用播放器模組設定播放器廣告文件。

    6. 核取「啟用伺服器端 (SSAI) 」核取方塊。

      除非您的帳戶已啟用 SSAI，否則伺服器端選項將不會出現。
    7. 從 [選取組態] 下拉式功能表中，選取您要與此播放器建立關聯的廣告設定。

    8. 如果您希望覆蓋在廣告上顯示，請勾選 [啟用廣告資訊覆蓋 ] 核取方塊。這包括「進一步了解」和廣告倒數疊加層。

      在您在此啟用並在 VAST 廣告代碼中進行設定之前，覆蓋層才會顯示。
    9. 選取「儲存」按鈕。

    10. 在左側導覽功能表中，選取 [ JSON 編輯器 ]。

    11. 在 JSON 編輯器中，向下捲動，直到您看到ad_failover: true屬性為止。

    12. 使用下列項目取代ad_failover: true屬性： 

      "ima_preroll_with_ssai": true

    13. 您的 JSON 編輯器應該看起來像這樣：

    14. 選取「儲存」按鈕。

    15. 若要發佈播放程式，請選取 [ 發佈和嵌入 > 發佈變更 ]。

    16. 現在，您已準備好發佈您的現場活動。如需詳細資訊，請參閱即時模組文件中的實作伺服器端廣告。
