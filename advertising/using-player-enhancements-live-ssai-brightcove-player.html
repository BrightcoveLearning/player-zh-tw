在 Brightcove Player 中使用 Live SSAI 的播放器增強功能

{{ page.description }}

介紹

Brightcove的Live SSAI播放器增強功能可讓您通過廣告計數，廣告倒數計時器，點擊廣告和隨播廣告來改善實時流中的廣告間隔。可以使用Brightcove Player播放這些實時流。

如果您不熟悉此功能，請參閱以下內容：

  • 概述：Live SSAI 的播放器增強功能
  • 實施實時SSAI的播放器增強

腳步

要使用Brightcove Player播放具有播放器增強功能的實時流，請按照下列步驟操作：

  1. 從Video Cloud Studio的創建或選擇現有播放器玩家們模塊。

  2. Brightcove Player需要使用SSAI插件進行配置。跟著添加SSAI插件步驟中所述以編程方式實施SSAI SSAI插件文檔的部分。

    對於 Live SSAI 的播放器增強功能，Brightcove Player 需要 SSAI 插件（videojs-ssai）。目前，您無法使用 Video Cloud Studio 將 SSAI 插件添加到您的播放器，因為它需要廣告配置 ID。
  3. 遵循實施實時SSAI的播放器增強文檔，直至創建回放令牌。
  4. 選擇一個實現：
    • 標準嵌入代碼
    • 進階嵌入程式碼
    • 實時SSAI來源
  5. 繼續中的工作流程實施實時SSAI的播放器增強文檔。

標準嵌入代碼

從Video Cloud Studio的視頻發布視頻時媒體模塊，您可以使用標準的嵌入代碼來播放直播。按著這些次序：

  1. 來自媒體模塊，選擇要進行實時流式傳輸的遠程視頻，然後使用支持SSAI的播放器進行發布。
  2. 複製標準的嵌入代碼並將其粘貼到您的網頁中。

  3. 在裡面<iframe>標籤，您應該會看到一個src屬性。在此末尾，添加adConfigId具有您的值的網址參數播放令牌。

    <iframe src="https://players.brightcove.net/6027103979001/IqljF4Olr_default/
index.html?videoId=6089818309001&adConfigId=live.aaB61234aabbccdd567890"
  allowfullscreen=""
  allow="encrypted-media"
  width="960" height="540">
</iframe>
  4. 您的播放器已準備好播放您的直播。

進階嵌入程式碼

從Video Cloud Studio的視頻發布視頻時媒體模塊，您可以使用高級嵌入代碼來播放直播。按著這些次序：

  1. 來自媒體模塊，選擇要進行實時流式傳輸的遠程視頻，然後使用支持SSAI的播放器進行發布。
  2. 複製高級嵌入代碼，然後將其粘貼到您的網頁中。

  3. 在裡面<video-js>標籤，添加一個data-ad-config-id具有您的值的屬性播放令牌。

    <video-js
  data-account="6027103979001"
  data-player="IqljF4Olr"
  data-embed="default"
  controls=""
  data-video-id="6089818309001"
  data-ad-config-id="live.aaB61234aabbccdd567890"
  data-playlist-id=""
  data-application-id=""
  width="960" height="540">
</video-js>
<script src="https://players.brightcove.net/6027103979001/IqljF4Olr_default/index.min.js"></script>
  4. 您的播放器已準備好播放您的直播。

實時SSAI來源

與VOD SSAI源不同，實時SSAI源不包括初始VMAP。這是因為實時SSAI流無法事先知道何時會包含廣告休息時間。取而代之的是，廣告間隔是由您添加到實時作業中的提示點觸發的。

Brightcove Live SSAI流以具有特定命名方案的標準HLS清單URL開頭：

https://.../playlist_ssaiM.m3u8

SSAI插件會查找_ssaiM.m3u8後綴和普通的HLS MIME類型

從Video Cloud Studio的視頻發布視頻時媒體模塊，您可以按以下方式播放實時SSAI流：

  1. 來自媒體模塊，選擇要進行實時流式傳輸的遠程視頻，然後使用支持SSAI的播放器進行發布。
  2. 複製高級嵌入代碼，然後將其粘貼到您的網頁中。

  3. 在頁面代碼中，添加以下代碼：

    <video-js id="myPlayerID"
  data-account="6027103979001"
  data-player="IqljF4Olr"
  data-embed="default"
  controls=""
  data-video-id=""
  data-playlist-id=""
  data-application-id=""
  width="960" height="540">
</video-js>
<script src="https://players.brightcove.net/6027103979001/IqljF4Olr_default/index.min.js"></script>
<script>
  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;
    myPlayer.src({
      type: 'application/x-mpegURL',
      src: 'https://path/to/playlist_ssaiM.m3u8'
    })
  });
</script>
  4. 您的播放器已準備好播放您的直播。
