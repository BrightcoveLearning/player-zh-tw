提供此示例和相關代碼作為生產開發的指南。該代碼未經認證可用於生產用途，也未經官方支持。

開始播放其中一個視頻。接下來，在其他播放器中播放視頻，然後在另一個正在播放的播放器中看到視頻停止播放。

在某些情況下，您可能希望在單個網頁上有多個播放器，如本示例所示，但通常在加載多個視頻的頁面上使用單個播放器會帶來更好的用戶體驗。

申請流程

此應用程序背後的基本邏輯是：

  • 確定頁面上有多少個播放器。
  • 循環播放播放器，將它們初始化為Brightcove播放器，並設置一個play每個事件監聽器。
  • 每次play處理事件後，循環播放器並暫停所有沒有調度該事件的播放器名稱的播放器。

在播放器上創建循環，並為每個播放器分配一個事件處理程序

找到標有以下代碼的代碼：

// ### Determine the available player IDs ###

如果頁面上有多個播放器，它們將存儲在對象名稱中videojs.players按鍵是玩家的名字。您使用JavaScript Object.keys()方法提取循環所需的播放器名稱，而不是實際的播放器對象。在循環中，您分配一個事件處理程序，然後將播放器推入數組。

處理play事件並停止其他玩家玩

找到標有以下代碼的代碼：

// ### Handle all players' play event ###

每當處理播放事件時，您都將從事件對像中提取播放器名稱（id），然後循環播放所有播放器並暫停所有播放器，然後使用沒有剛剛開始播放視頻的播放器的名稱。
