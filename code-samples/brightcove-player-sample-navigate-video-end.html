播放器示例

提供此示例和相關代碼作為生產開發的指南。該代碼未經認證可用於生產用途，也未經官方支持。

播放完非常短的視頻後，您將看到一個疊加層，其中包含指向Brightcove主頁的鏈接。使用Brightcove提供的Overlay插件，因此不需要JavaScript。CSS用於設置疊加層的樣式。

在這種情況下不會顯示 iframe 實現，因為新的 URL 頁面將包含在 iframe 播放器的維度內，很可能不是當時想要的行為。您將在下面的 CodePen 中看到這種奇怪的行為。

應用程式樣式

CSS用於更改疊加層的外觀。在此示例中，使用的CSS是：

/* override default styles for overlay */
.video-js .vjs-overlay {
    width: 100%;
    height: 20%;
    color: #000;
    background-color: rgba(255, 255, 255, 0.8);
    left: 0;
    top: 0;
    padding-top: 5%;
    padding-bottom: 5%;
    margin-top: 10%;
    padding-left: 0;
}

您需要更改CSS以定義所需的外觀，然後將CSS保存在Internet上可訪問的位置。您將在插件配置中指向此CSS。

播放程式組態

在這種情況下，您將使用Studio添加顯示覆蓋插件。

  1. 打開球員模塊並創建一個新播放器或找到您希望添加插件的播放器。
  2. 單擊播放器的鏈接以打開播放器的屬性。
  3. 點擊插件在左側導航菜單中。
  4. 接下來，點擊添加插件 .
  5. 為了插件名稱進入overlay。
  6. 為了JavaScript網址，輸入： 
    https://players.brightcove.net/videojs-overlay/2/videojs-overlay.min.js
  7. 為了的CSS URL，輸入： 
    https://players.brightcove.net/videojs-overlay/2/videojs-overlay.css
  8. 在配置選項中輸入選項(JSON)文本框。 
    {
  "content": "<strong>Default overlay content</strong>",
  "overlays": [
    {
      "content": "<a href='http://www.brightcove.com'><span style='font-size:large'><strong>Brightcove Homepage</strong></span></a>",
      "start": "ended",
      "end": "play"
    }
  ]
}
  9. 點擊節省 .
  10. 若要發佈播放程式，請按一下「發佈與內嵌」>「發佈變更」。
  11. 若要關閉開啟對話方塊，請按一下「關閉」。
有關使用 Studio 為玩家分配插件的更多信息，請參閱配置播放器插件文檔。
