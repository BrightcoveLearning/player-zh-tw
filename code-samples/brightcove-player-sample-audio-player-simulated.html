開發順序

這是推薦的開發順序：

  1. 使用頁內嵌入播放器實現來測試播放器，插件和CSS的功能（如果需要CSS）
  2. 將插件的JavaScript和CSS放入單獨的文件中以進行本地測試
  3. 解決任何錯誤後，將插件代碼和CSS部署到服務器上
  4. 使用Studio將插件和CSS添加到播放器
  5. 如果您確定iframe實現更合適，請替換頁內嵌入播放器實現（在下一部分中詳細介紹）

有關這些步驟的詳細信息，請查看一步步：插件開發指導。

iframe或頁內嵌入

在開發Brightcove Player的增強功能時，您需要確定代碼是否最適合iframe或頁內嵌入實施。最佳實踐建議是構建一個可與iframe實現配合使用的插件。使用iframe播放器的優點是：

  • 與現有的JavaScript和/或CSS無衝突
  • 自動響應
  • iframe可簡化在社交媒體應用中的使用（或需要將視頻“傳播"到其他應用中的任何時候）

儘管集成頁內嵌入播放器可能會更加複雜，但是有時您會圍繞該實現計劃代碼。概括地說，當包含頁面需要與播放器進行通信時，這種方法是最好的。具體來說，以下是一些示例：

  • 包含頁面中的代碼需要偵聽播放器事件並對其進行操作
  • 播放器使用包含頁面中的樣式
  • iframe將導致應用邏輯失敗，例如從包含頁面的重定向

即使最終實現未使用iframe代碼，您仍可以將頁內嵌入代碼與JavaScript插件和CSS單獨文件一起使用。這封裝了你的邏輯，以便您可以輕鬆地在多個玩家中使用它。

選擇實施

儘管此處描述的播放器可以在嵌入的iframe中使用，但是有一些CSS應用於播放器，可以使角落變圓並保持控件可見。這種樣式取決於具有ID的播放器，並且只能與頁面內嵌入一起使用。在此使用頁內嵌入。

創建播放器

要創建一個播放器並為其分配視頻，請按照下列步驟操作：

  1. 在裡面球員模塊，創建一個新的播放器或使用現有的播放器。
  2. 編輯播放器設置以更改大小：
    • 選擇固定尺寸
    • 選擇自定義的寬高比
    • 將寬度設置為您喜歡的任何值，但是將高度至80像素
  3. 在裡面媒體模塊，選擇一個純音頻項目並與該玩家一起發布。將純音頻內容分配給播放器並發布。
  4. 複製頁內嵌入代碼，然後將其粘貼到新的HTML文件中。
  5. 添加id僅音頻給玩家。
如需如何取得此內嵌程式碼的詳細資訊，請參閱逐步說明：布萊特灣玩家文件

添加自定義CSS

現在，在頁面頂部的樣式欄中添加一些自定義CSS。樣式將做三件事：

  • 刪除大播放按鈕上的背景（因為它對於播放器的高度來說太大了）
  • 即使用戶將鼠標移開，在播放過程中也要保持控制欄可見
  • 圓角播放器，使其看起來更像音頻播放器

您需要添加的CSS代碼如下所示：

/* keep the controls visible during playback */
#audioOnly .vjs-control-bar {
    transform: none;
    background-color: transparent;
}
/* round the corners of the player */
#audioOnly {
    border-radius: 1em;
    width: 480px;
    height: 80px;
}
/* remove the background from the big play button */
.video-js.vjs-mouse .vjs-big-play-button {
    background: none;
}
/* round the corners of the dock */
#audioOnly .vjs-dock-text {
    border-radius: 1em 1em 0 0;
}
