開發順序

這是推薦的開發順序：

  1. 使用頁內嵌入播放器實現來測試播放器，插件和CSS的功能（如果需要CSS）
  2. 將插件的JavaScript和CSS放入單獨的文件中以進行本地測試
  3. 解決任何錯誤後，將插件代碼和CSS部署到服務器上
  4. 使用Studio將插件和CSS添加到播放器
  5. 如果您確定iframe實現更合適，請替換頁內嵌入播放器實現（在下一部分中詳細介紹）

有關這些步驟的詳細信息，請查看一步步：插件開發指導。

iframe或頁內嵌入

在開發Brightcove Player的增強功能時，您需要確定代碼是否最適合iframe或頁內嵌入實施。最佳實踐建議是構建一個可與iframe實現配合使用的插件。使用iframe播放器的優點是：

  • 與現有的JavaScript和/或CSS無衝突
  • 自動響應
  • iframe可簡化在社交媒體應用中的使用（或需要將視頻“傳播"到其他應用中的任何時候）

儘管集成頁內嵌入播放器可能會更加複雜，但是有時您會圍繞該實現計劃代碼。概括地說，當包含頁面需要與播放器進行通信時，這種方法是最好的。具體來說，以下是一些示例：

  • 包含頁面中的代碼需要偵聽播放器事件並對其進行操作
  • 播放器使用包含頁面中的樣式
  • iframe將導致應用邏輯失敗，例如從包含頁面的重定向

即使最終實現未使用iframe代碼，您仍可以將頁內嵌入代碼與JavaScript插件和CSS單獨文件一起使用。這封裝了你的邏輯，以便您可以輕鬆地在多個玩家中使用它。

其他HTML

沒有其他HTML元素添加到頁面。

申請流程

此應用程序背後的基本邏輯是：

  • 建立HTML <div>標記來保存按鈕。
  • 將ID分配給<div>標籤。
  • 將按鈕的圖像分配給<div>標籤。
  • 插入<div>標籤進入控制欄。
  • 添加click的事件處理程序<div>標籤以實際更改播放。

建立HTML <div>標籤

找到標有以下代碼的代碼：

// ### Create divs for buttons ###

二<div>標籤被創建，每個按鈕一個。

將ID分配給<div>標籤

找到標有以下代碼的代碼：

// ### Assign IDs for later element manipulation ###

為每個ID分配一個ID <div>標記，以便可以對其進行尋址，並可以稍後在代碼中對其進行動態更改。

將按鈕的圖像分配給<div>標籤

找到標有以下代碼的代碼：

// ### Assign properties to elements and assign to parents ###

適當的圖像被動態分配給<img>標籤，然後將這些圖像分配給<div>標籤。

插入<div>標籤進入控制欄

找到標有以下代碼的代碼：

// ### Get controlbar and insert elements ###

使用$()這行代碼中的方法（與jQuery無關係）：

controlBar = myPlayer.$('.vjs-control-bar');

這等效於使用以下JavaScript：

document.getElementsByClassName('vjs-control-bar')[0];

的優勢$()方法是可以將其作用於播放器，因此多個播放器不會引起問題。它還返回單個元素，而不是數組。

基本上$ = querySelector和$$ = querySelectorAll .它們是簡寫myPlayer.el().querySelector和myPlayer.el().querySelectorAll .

引用控制欄後，將獲得對包含音量控件的元素的引用，作為放置按鈕的引用。最後，兩者<div>標籤被插入到控制欄中。

添加事件處理程序

找到標有以下代碼的代碼：

// ### Add event handlers to jump back or forward ###

事件處理程序的邏輯非常簡單。唯一的細節是確保時間未在一端設置為負值，或者在另一端未設置為持續時間以上。

應用程式樣式

CSS控製圖片在圖片中的大小和位置<div>標籤。

插件代碼

通常，將JavaScript轉換為Brightcove Player插件時，需要進行名義上的更改。一項必需的更改是替換標準的使用ready()方法與定義插件的代碼。

這是可以與播放器一起使用的非常常用的JavaScript代碼開始：

videojs.getPlayer('myPlayerID').ready(function() {
var myPlayer = this;
});

您將更改第一行以使用標準語法來啟動Brightcove Player插件：

videojs.registerPlugin('pluginName', function(options) {
var myPlayer = this;
...
});

如前所述，您可以在該文檔的相應GitHub存儲庫中看到該插件的JavaScript代碼： back-forward-buttons.js。
