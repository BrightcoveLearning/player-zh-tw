範例播放程式

以下範例影片示範如何使用 360°/VR 外掛程式。按一下影片播放後，您可以使用滑鼠或方向鍵來控制桌面上的檢視，或是在行動裝置上檢視陀螺儀。360° 影片標誌播放按鈕會在播放時消失。

 

簡介

功能

以下是 360°/VR 外掛程式的功能：

  • 行動裝置上的陀螺儀整合：旋轉手機時，透視會隨著您的手機而改變。
  • 支援紙板/虛擬實境模式：在支援此功能的裝置上，「紙板檢視器」圖示會出現在控制列中。按下時，投影將會調整為與紙板式 VR 護目鏡搭配使用。
  • 視訊雲端媒體 360° 設定：如果您正在使用視訊雲端影片，外掛程式會遵循視訊雲端在視訊雲端媒體上的 360° 設定。如果該值不存在，它將播放視頻作為一個普通視頻。您不需要確保使用 360°/VR 外掛程式的玩家只能播放 360° 影片。
  • 非視訊雲端媒體：如果您是 不使用視訊雲端提供的媒體，您仍然可以使用該插件。您將需要調用vr插件傳遞一個projection值設置如下的對象： 
    <script type="text/javascript">
  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;
    myPlayer.vr({projection: '360'});
  });
</script>
    將projection屬性設定為360，equirectangular或者Sphere在邏輯上都是等價的。

    當然，上面的代碼假設您已經包含了插件的 JavaScript 和 CSS 文件，如本文稍後所述。

瀏覽器相容性

  • 為了在所有瀏覽器中獲得最佳效能，您應該更喜歡使用此外掛程式的影片透過 HLS 的 MP4 轉譯。
  • 此外，在上傳影片時，Brightcove 建議您建立一個自訂的動態傳遞內嵌描述檔，包含單一 1080p 轉譯和至少一個音訊轉譯。

該插件與以下主流瀏覽器的最新版本兼容：

  • Chrome 桌面
  • 瀏覽器行動裝置 (安卓 6 個以上版本)
  • 火狐桌面
  • 邊
  • Safari 桌面 *
  • 野生動物園行動裝置 *
* 在 Safari 上使用外掛程式時，如果您使用的是非影片雲端影片 (未透過影片雲端擷取的影片)，請注意，您必須將設定屬性crossorigin設定為anonymous。

外掛程式不適用於網際網路總管。

實作概觀

實現插件很簡單，只需完成以下兩個步驟即可：

  • 正確地內嵌視訊。
  • 將外掛程式新增至播放器。

擷取 360° 視訊

注意：如需建立 360° 視訊的詳細資訊，請參閱使用 360° 視訊文件。

錄製 360° 影片後，您需要將其內嵌到雲端視訊。一點了解 360° 視訊的運作方式，將有助於您了解擷取設定檔選擇的一些原因。將 360° 影片拍攝，投影在球體內部。這表示播放器中視訊的可見區域只是整個視訊的一小部分。下面的截圖給你這個概念的一個粗略的想法。紅色矩形內部的區域是播放器中可見的區域，但整個視訊只有一小部分。

由於只有一小部分的視頻將在播放器中在播放器中，因此您希望播放一個非常高品質的轉譯，而不會降低到較低品質的轉譯。實質上，您正在「放大」視頻的一部分，質量差的轉譯會產生非常差的觀看體驗。因此，上傳影片時，Brightcove 建議您使用單一 1080p 轉譯和至少一個音訊轉譯來建立自訂的動態傳遞內嵌描述檔。使用 360°/VR 外掛程式時，這比 HLS 轉譯更好。

使用播放器模組實作

若要實作 360°/VR 外掛程式，您會將外掛程式名稱和網址新增至外掛程式的 JavaScript 和 CSS 檔案：

  1. 打開球員模塊並創建一個新播放器或找到您希望添加插件的播放器。
  2. 按一下播放程式的連結以開啟播放程式的屬性。
  3. 點擊插件在左側導航菜單中。
  4. 從新增外掛程式下拉式清單中，選取自訂外掛程式。
  5. 在「外掛程式名稱」中輸入vr。
  6. 在這個網址中，輸入： 
    https://players.brightcove.net/videojs-vr/1/videojs-vr.min.js
  7. 對於 CSS 網址，請輸入： 
    https://players.brightcove.net/videojs-vr/1/videojs-vr.css
  8. 您不需要為此外掛程式輸入任何選項。
  9. 按一下「儲存」。
  10. 點擊造型在左側導航菜單中。
  11. 如果您覺得這會是更好的觀賞體驗，您可以停用標題和說明碼頭。若要這麼做，請取消勾選 [ 顯示標題和說明] 核取方塊。
  12. 若要發佈播放程式，請按一下「發佈與內嵌」>「發佈變更」。
  13. 若要關閉開啟對話方塊，請按一下「關閉」。

選項

分型板

類型：boolean預設值：假

強制在所有裝置上顯示紙板按鈕。

運動控制

類型：boolean默認值：在 iOS 和安卓

確定是否應啟用運動/陀螺儀控制。

投影

類型：string預設值：自動

視訊類型 projection選項的值
球體 360 、圓球或等矩形
立方體 立方體或 360 立方體
不是 360° 視訊 無
如果您想檢查當前視頻是否通過player.mediainfo.projection 自動
並排 360° 視訊 360_LR
由上至下 360° 視訊 360_TB

播放器投影

類型：string預設值：未定義

此值應根據來源設定，以根據影片關閉 360° 視訊。

除錯

類型：boolean預設值：假

啟用此外掛程式的偵錯記錄。

廣告

此外掛程式對廣告的支援僅限於 IM3 預捲。目前不支援其他類型的廣告和其他廣告廠商。

已知問題

  • 不支援在同一播放器中混合 VR 和非 VR 內容。對於這種觀眾體驗的建議是根據需要移除並重新創建播放器。請參閱布萊特灣玩家範例：動態載入播放程式
  • 在 iOS 13 上，應用程序將要求用戶允許使用設備運動和方向（陀螺儀）功能。開發人員必須使用權限 API 來實作此功能。如需詳細資訊，請參閱 Apple 的「要求許可」文件。
  • 在 Safari 12.x 中，行動裝置的陀螺儀整合預設為關閉。必須變更「動作與方向存取」設定，才能使陀螺功能正常運作。
  • 雖然 360°/VR 外掛程式可能適用於直播，但尚未正式支援。
  • 最新的 360°/VR 外掛程式與亮灣播放器 6.17.0 及更新版本相容。
  • 由於 360°/VR 外掛程式使用 MP4 轉譯功能，因此不支援多個音軌 (因為瀏覽器不支援使用 MP3 的多個音軌)。您可以使用內嵌不包含 MP4 轉譯的設定檔，以避免這個問題，針對具有多個音軌的 360° 視訊。
  • 360°/VR 外掛程式無法與網際網路總管運作。
  • 360°/VR 外掛程式不支援 DRM。
  • 外掛程式不支援立體/堆疊影片。
  • 在移動設備上，只有水平滑動工作（這會水平改變視圖），但垂直滑動屏幕不會執行任何操作。垂直滑動 適用於 桌面瀏覽器。
  • 布萊特灣玩家的標準（iframe）實施不會改變方向。
  • 在 iOS 上，為了讓 360° 插件正確渲染視頻，播放前預加載視頻設置必須設置為預加載視頻（最快）在播放器中：
  • 在 iOS 上，全螢幕模式按鈕已停用，因為 360°/VR 外掛程式無法與 iOS 原生播放程式搭配使用。因此，建議您設定非全螢幕視訊播放的playsinline屬性，如 iOS 和 Brightcove Player 文件所示。
