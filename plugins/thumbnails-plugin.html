概述

視頻縮略圖使用戶可以快速掃描進度條中他們感興趣的部分。當用戶將鼠標懸停在進度欄上時，Brightcove Player將顯示縮略圖。單擊進度條會將其帶到視頻中的該位置。

默認情況下，所有新玩家都啟用縮略圖搜索。
什麼時候生成圖像？

Brightcove可根據要求動態生成圖像。使用動態投放方式提取的任何視頻都將在第一個請求時生成圖像。這可確保不會為未使用的視頻創建圖像。生成後，就像其他任何圖像一樣，圖像會在本地和CDN上緩存以供重複查看。

視頻的圖像頻率是多少？

縮略圖列表根據初始化時播放器的像素寬度向下過濾。此列表根據播放器的樣式，窗口大小，平台/設備等而有很大不同。縮略圖插件旨在在用戶將鼠標懸停在進度欄上時大約每30像素顯示一個新的縮略圖。例如，如果播放器的寬度為1500像素，則播放器將嘗試將列表過濾為50個縮略圖。WebVTT提示的剩餘時間間隔將根據內容的持續時間進行調整。

筆記：
  • 當前使用播放列表播放器時，此功能不可用
  • 不支持移動設備上的縮略圖

需求

以下要求適用於縮略圖插件：

  • 您必須使用Brightcove Player版本6.41.0+
  • 必須提取視頻才能進行動態投放
  • 此功能需要使用Playback API v2
  • 對於服務器端廣告插入（SSAI），您必須使用videojs-ssai插件版本1.10.0+

入門

任何擁有 Brightcove Video Cloud 帳戶的人都可以使用此功能。

確保為動態投放提取了您的視頻。

使用Studio實施縮略圖

配置播放器以進行縮略圖搜索的最簡單方法是使用Video Cloud Studio。

  1. 打開選手們模塊。使用現有播放器或創建一個新播放器。

  2. 選擇播放器鏈接以打開播放器的屬性。
  3. 在左側導航中，選擇控制項。

  4. 查看縮略圖搜尋。

  5. 要發布播放器，請選擇發布和嵌入>發布更改

  6. 現在已為您的播放器配置了縮略圖插件。

    使用為動態投放而提取的視頻，然後將其與我們在上述步驟中配置的播放器一起發布。將鼠標懸停在播放器的進度條上時，應該會看到縮略圖。

實施WebVTT縮略圖

如果您選擇使用自己的縮略圖，則可以創建一個自定義WebVTT文件，然後使用player.addRemoteTextTrack()方法。步驟如下：

  1. 創建一個.vtt遵循標準WebVTT格式的文件。將標題文本替換為每個縮略圖的URL。它可能看起來像這樣：

  2. 在Video Cloud Studio中，選擇為動態交付而攝取的視頻。複製版本為6.41.0+的播放器的“高級嵌入"代碼。
  3. 在網頁的HTML文件中，粘貼“高級嵌入"代碼。
  4. 在裡面head HTML代碼部分添加縮略圖CSS文件： 
    //players.brightcove.net/videojs-thumbnails/1/videojs-thumbnails.css
  5. 在裡面body HTML代碼部分添加縮略圖JavaScript文件： 
    //players.brightcove.net/videojs-thumbnails/1/videojs-thumbnails.js
  6. 到video標籤，添加一個id財產。
  7. 收盤前body標籤，添加一個script堵塞。

  8. 在剛創建的腳本塊中，添加以下內容：

    財產 價值
    type 視頻的啞劇類型
    src 視頻的源URL

    您的代碼應與此類似： 

    var player = bc('myPlayerID');

player.thumbnails();

player.ready(() => {
  player.src({
    type: 'video/mp4',
    src: '//solutions.brightcove.com/bcls/videos/Great Blue Heron.mp4'
  });
});

  9. 在同一腳本塊中，添加以下內容：

    插件初始化且播放器準備就緒後，您將使用WebVTT文件將縮略圖源傳遞到WebVTT文件中的插件。addRemoteTextTrack()方法。使用縮略圖文件調用方法時，還必須設置：

    財產 價值
    src 帶縮略圖的WebVTT文件的位置
    kind 調成metadata
    label 調成thumbnails
    mode 調成hidden
    addRemoteTextTrack() manualCleanup 調成false以便在源更改時自動刪除音軌

    您的代碼應與此類似： 

    var player = bc('myPlayerID');

player.thumbnails();

  player.ready(() => {
    player.src({
      type: 'video/mp4',
      src: '//solutions.brightcove.com/bcls/videos/Great Blue Heron.mp4'
    });

  player.addRemoteTextTrack({
    src: 'thumbnails.vtt',
    kind: 'metadata',
    label: 'thumbnails',
    mode: 'hidden'
  }, false);
});
