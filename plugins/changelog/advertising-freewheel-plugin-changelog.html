注意：正如實際的 Freewheel 插件文檔中提到的那樣，在使用正確的 SDK URL 部分中，Freewheel 消除了動態 SDK URL，以支持版本化的 URL。這表示 Brightcove Player 外掛程式現在預設會鎖定為特定版本，而不是最新版本。布萊特灣將嘗試跟上 Freewheel SDK 的版本，但可能會落後。

您可能想要在他們的 Brightcove 播放器集成中管理自己的 SDK 版本。這涉及將Html5.sdkurl選項傳遞給 Freewheel 插件：

player.FreeWheelPlugin({
  Html5: {
    sdkurl: '  preferred-sdk-url'
  }
});
