將數據傳遞到插件

{{ page.description }}

定義選項屬性

調用插件時，您還可以使用其他插件options子財產plugins。這options屬性必須是一個對象，但可以包含複雜的數據結構，例如對像數組。

這是curl語句的一部分，顯示了帶有必需屬性的plugins屬性name子屬性和可選options子財產。傳入的數據options然後可以在插件中使用屬性。

"plugins": [{
  "name": "navigateOnVideoEnd",
  "options" : {"redirectURL": "http://docs.brightcove.com"}
}]

您還可以在使用embed_in_page播放器實現，播放器標記位於HTML頁面中，而不位於iframe中。為此，只需構建數據對象，然後將其作為參數傳遞。這裡顯示一個示例：

<script type="text/javascript">
  var options = {"redirectURL": "http://docs.brightcove.com"};
</script>
<script>videojs.getPlayer('myPlayerID').navigateOnVideoEnd(options);</script>

使用選項數據

這options屬性使您能夠通過播放器配置將數據傳遞到插件，以在插件初始化時使用。例如，使用redirectURL上面顯示的對象，您將在插件本身中執行以下操作：

videojs.registerPlugin('navigateOnVideoEnd', function (options) {
  var myPlayer = this;
  myPlayer.on("ended", function () {
    window.location.href = options.redirectURL;
  });
});

注意：您使用標準插件實現，但使用options作為匿名函數中的參數。然後，您可以使用訪問該值options.propertyName符號。

如前所述，options屬性可以是所需的任何數據結構，例如對像數組。請記住，如果您傳入一個對像數組，則必須以對像數組的形式訪問數據。
