定義

下表給出了關鍵組件的定義：

元件 描述
色調 在視頻開始播放之前顯示的圖像
文字追蹤顯示 顯示為用於字幕和字幕的單獨軌道的文本（WebVTT格式）
載入微調器 一直顯示動畫的旋轉輪，直到加載視頻
大按鈕 加載時顯示在按鈕區域內的播放按鈕
控制列 所有主要播放器控件的容器
    播放切換 在播放和暫停之間切換的播放按鈕
    靜音切換 在音頻靜音和不靜音之間切換的按鈕
        音量控制 滑桿控制音量
            音量等級 將鼠標懸停在音量欄上時顯示當前音量
                音量手柄 顯示音量水平的當前位置，可以拖動以調節音量水平
    當前時間顯示 當前播放視頻的持續時間，顯示為mm：ss
    時分器 正斜杠'/分離當前時長和視頻時長
    持續時間顯示 影片的總時長
    進度控制 包含搜索、加載進度和播放進度條的組件
您可以使用以下方法關閉查看器與 progressControl 及其子項的交互： 
player.controlBar.progressControl.disable();
        搜尋欄 媒體欄在接收到已提取媒體的事件後變為活動狀態
         LoadProgressBar 指示正在提取的視頻的進度
         PlayProgressBar 指示正在播放的視頻的進度
        搜索句柄 顯示播放過程中播放頭的當前位置，可以拖動以調整播放頭
    現場控制 顯示單詞居住如果直播直播
    剩餘時間顯示 顯示尚未播放的視頻的時長
    墊片 元素插入到ControlBar中，特別是用於添加圖標
    播放率 顯示播放速率
    章節按鈕 按鈕顯示章節
    字幕按鈕 切換以顯示字幕設置表單
    全屏切換 切換全屏顯示的按鈕
字幕設置 用於顯示和更改字幕設置的表格
DockText 標題和說明的容器
    碼頭名稱 影片標題
    碼頭說明 影片的簡短說明
塢架 由社交媒體插件使用

班級信息

播放器類和所有控件類均從Component類或Component的子類繼承。

videojs.Control = videojs.Component.extend();
videojs.Button = videojs.Control.extend();
videojs.PlayToggle = videojs.Button.extend();

UI組件體系結構使將子組件添加到父組件並建立整個用戶界面（如Video.js播放器的控件）變得更加容易。添加子組件時，它將子組件插入到父組件中，並允許您根據需要操縱該元素，如下所示：

myButton = myPlayer.controlBar.addChild('Button');
myButton.addClass('bc-download-button');
myButton.on('click',function(){
  ...
})
