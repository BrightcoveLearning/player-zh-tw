獲取憑證

要使用此配置報告應用程序，您將需要適當的憑據。

在大多數情況下，獲取憑據的最簡單方法是通過 Studio Admin API 身份驗證部分（需要您帳戶的管理員權限）。看管理 API 身份驗證憑據詳情。您將獲得以下單一權限：

如果Studio中沒有所需的權限，或者您希望直接從OAuth API獲取權限，請選擇獲取客戶憑證下面列出的文件。無論您選擇哪個選項，您都需要請求正確的操作權限。以下內容可與cURL或Postman一起使用，以獲取播放器管理API的所有權限：

  "operations": [
    "video-cloud/players/read"
  ]
  • OAuth：使用cURL獲取客戶端憑據
  • OAuth：使用獲取客戶端憑據郵差

請注意以下事項：

  • 同樣在插件報告中，以開頭的插件@ brightcove /是Brightcove提供的插件。其他的要么是自定義插件，要么是通過Studio或curl手動加載的Brightcove提供的插件。
