Players 模塊支持 Google 的 Universal Analytics 數據模型。當前，不支持較新的Google Analytics（分析）4更新。Brightcove 建議在與 Players 模塊和 Brightcove Players 集成時使用 Universal Analytics。

介紹

配置過程包括以下任務：

  • 從Brightcove Player支持的事件中選擇要跟踪的事件。
  • 從Google Analytics（分析）插件支持的維度中選擇您要使用的自定義維度。
  • 在Google Analytics（分析）中，創建所選的自定義維度。
  • 在Video Cloud Studio中，添加Google Analytics（分析）插件的配置信息。您將使用播放器的JSON編輯器執行此操作。

選擇活動

在JSON配置中，您將列出要跟踪的事件。確定您要跟踪以下哪項：

  • player_load
  • video_engagement
  • 視頻印象
  • play_request
  • ad_start
  • ad_end
  • 錯誤
  • 視頻視圖
  • video_complete

選擇自定義尺寸

您可以選擇八個自定義維度，以在Google Analytics（分析）數據中使用。確定要使用以下哪個：

  • bcvideo_account
  • bcvideo_platform_version
  • bcvideo_player
  • bcvideo_range
  • bcvideo_session
  • bcvideo_tags
  • bcvideo_video_duration
  • bcvideo_video_seconds_viewed

創建自定義尺寸

您必須創建要使用的每個自定義維度。Google提供的本文檔中的步驟將引導您完成此過程：創建和編輯自定義維度和指標。

在創建每個自定義維度的過程中，系統會要求您選擇一個範圍。對於這些自定義維度，請使用值 打.同樣，當您創建每個自定義維度時，索引或序列，數字也會與之相關聯，如以下屏幕截圖所示：

您將需要提供創建的自定義維度表，因為配置JSON中需要索引號。

創建尺寸時，可以選擇一個更友好的名稱。例如，考慮以下內容：

  • 創建自定義維度時bcvideo_video_duration你簡單地命名期間因為您希望在報告中使用更友好的名稱。
  • 創建時，與關聯的索引期間是7 .
  • 在配置JSON中，使用以下分配： 
    "bcvideo_video_duration": "dimension7"
    索引號將友好名稱與插件的內部名稱相關聯。

將JSON配置添加到播放器

您已決定要使用的事件以及要使用的自定義維度，並創建了後者。您已經準備好構建將包含在播放器配置中的JSON。

下面顯示了一個示例，並帶有一個表，解釋了每個邏輯代碼組。您可以按照本指南構建自己的JSON。

配置JSON完成後，您可以按照以下步驟將其粘貼到Brightcove Player的配置中：

  1. 登錄到Video Cloud Studio。
  2. 導航到選手們模塊，然後單擊要向其添加新配置的播放器。
  3. 在左側導航中，點擊JSON編輯器。
  4. 如果已經為播放器配置了插件，則可以將代碼粘貼到該部分中。否則，您將需要為插件添加一些JSON。無論哪種情況，結果都應該類似於以下內容，但是當然要使用您的播放器信息和JSON（該插件的JSON配置以黃色突出顯示）： 
    { "compatibility": true, "video_cloud": { "policy_key": "BCpkADawqM2w1K7269Ic17hjK0O2-ibj221pVcELpF1l5nZuiL-123456789123456789" }，“玩家”：{ "template": { "name": "single-video-template", "version": "6.48.5" } }, "插件": [ { "registry_id": "@brightcove/videojs-bc-ga", "version": "1.x", "options": { "paramsToTrack": { "bcvideo_video_seconds_viewed": "dimension1", "bcvideo_range": "dimension2", "bcvideo_video_duration": "dimension3", "bcvideo_player": "dimension4", "bcvideo_session": "dimension5" } , "eventsToTrack": { "video_impression": "Video Impression", "play_request": "Play Request", "video_engagement": "Video Engagement" }，“跟踪器”："UA-123456-1" } } ] }
  5. 保存更改並發布播放器，您的Google Analytics（分析）插件將可以使用。
