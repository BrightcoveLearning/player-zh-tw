音量控制

{{ page.description }}
注意：下面的示例使用volume()和muted()播放器準備就緒時執行這些方法。您無需等待metatdataloaded事件，例如，因為音量是播放器設置並且與正在播放的視頻無關。

volume（）方法

這volume()根據是否傳遞參數，方法既充當獲取器，也充當設置器。如果該方法用作設置器，則參數是介於0（靜音）和1.0（最大音量）之間的十進制數。參數是用十進制數表示的百分比。

以下是使用volume()將音量設置為60％的方法。

  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;
    myPlayer.volume(.6);
  });

靜音（）方法

這muted()根據是否傳遞參數，該方法既充當getter，又充當setter。如果該方法用作設置器，則參數為布爾值，true靜音和false取消靜音。

以下是使用muted()使播放器音頻靜音的方法。

  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;
    myPlayer.muted(true);
  });
注意：兩者之間有細微的差別volume(0)和muted(true)。所不同的是，切換靜音後，音量將保持之前的百分比muted()被使用了。當然，如果音量設置為 0，要恢復音頻，必須再次設置百分比。

成交量變化事件

如果您想對音量的變化做出反應，可以使用volumechange事件。您可以按以下方式設置事件偵聽器：

    myPlayer.on('volumechange', function( evt ){
      console.log('event: ', evt);
    })
