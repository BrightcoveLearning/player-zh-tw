播放器屬性

Brightcove Player屬性包含有關您的特定播放器的信息。有關設置播放器外觀和版本的詳細信息，請參見皮膚和播放器版本文檔。

版本

這bc.VERSION屬性指定播放器配置中定義的播放器版本。

這是一個例子：

    <video-js id="video_1" style="width: 640px; height: 360px;"
    data-video-id="4029697544001"
    data-account="1752604059001"
    data-player="default"
    data-embed="default"
    class="video-js"
    controls></video-js>
    <script src="https://players.brightcove.net/1752604059001/default_default/index.min.js"></script>


    <!-- custom script -->
    <script type="text/JavaScript">
        var myPlayer;

	videojs.getPlayer('video_1').ready(function() {
            myPlayer = this;
            console.log(bc.VERSION);
            console.log(bc.SKIN);
        });
    </script>
返回值：

在這種情況下返回的值是5.1.7

此值來自播放器配置，可以通過在瀏覽器中預覽播放器並更改來找到它。index.html至config.json。

皮膚

這bc.SKIN屬性指定應用於播放器的皮膚。

這是一個例子：

    <video-js id="video_1" style="width: 640px; height: 360px;"
    data-video-id="4029697544001"
    data-account="1752604059001"
    data-player="default"
    data-embed="default"
    class="video-js"
    controls></video-js>
    <script src="//players.brightcove.net/1752604059001/default_default/index.min.js"></script>

    <!-- custom script -->
    <script type="text/JavaScript">
        var myPlayer;

	  videojs.getPlayer('video_1').ready(function() {
            myPlayer = this;
            console.log(bc.VERSION);
            console.log(bc.SKIN);
        });
    </script>
返回值：

在這種情況下返回的值是空值，這表示Luna皮膚已應用到該播放器。這是來自的返回值的摘要bc.SKIN財產：

bc.SKIN值 應用皮膚
錯誤的 沒有皮膚，因此您可以使用自己的樣式覆蓋
石墨 石墨皮
所有其他值（或缺少值），包括undefined，null，true等。 月神皮膚（默認皮膚）

此值來自播放器配置，可以通過在瀏覽器中預覽播放器並更改來找到它。index.html至config.json。

注意：這bc.SKIN僅當"skin"播放器配置中的property具有顯式值。否則，它將是未定義的。
