概述

該頁面供潛在和當前的客戶執行測試。以下是您可以測試的一些選項：

  • HTML模式與Flash模式
  • 月神皮膚與石墨皮膚
  • 不同的視頻格式類型

播放器參考頁面位於http://players.brightcove.net/3303963094001/player-reference-app/index.html。

影片示範

了解播放器參考頁的一種好方法是觀看以下4 1/2分鐘的視頻：

提供視頻源

您需要做的第一件事是為播放器提供視頻源，或使用提供的默認視頻之一。您粘貼URL，然後在“配置"按鈕欄中單擊要測試的選項。您必須預先選擇選項，然後單擊加載。

播放期間在配置按鈕欄中選擇選項無效。您必須單擊配置按鈕，然後單擊加載再次。另外，您需要手動選擇正確的URL內容類型用於提供的視頻。如果您選擇了錯誤的類型，要么您會收到錯誤消息，要么視頻無法播放。

如果需要，可以使用視頻雲源進行測試。只需轉到媒體 Studio 中的模塊，單擊視頻，然後在視頻文件部分右鍵單擊一個條目，然後選擇複製鏈接 .然後，您可以將復制的鏈接粘貼到URL輸入區域中。

提供 DASH 視頻源時，您還需要提供許可證服務器 URL。請注意，此時您無法提供完整的鍵系統選項對象為玩家選項 .

提供播放器選項

需要一些說明的“配置"按鈕欄上的一個選項是播放器選項按鈕。

通過此按鈕，您可以使用有效的JSON格式數據將某些配置選項傳遞給播放器。例如，以下內容將提供一個新的海報，並在完成後進行視頻循環：

{ "poster": "http://solutions.brightcove.com/fish_poster.png", "loop": true }

這高級（頁內）嵌入代碼的可用選項文檔提供了您可以使用的選項。注意width和height選項將無法正確用作播放器選項，因為它們已被頁面的配置覆蓋。

查看顯示的數據

頁面上報告的數據通常很容易解釋，這裡將對兩個區域進行一些解釋，即比特率切換和活動時間表部分。

比特率切換

首先，如果您正在查看MP4視頻源，則本節將看不到任何信息。對於MP4，將發送視頻源，並且沒有進一步的協商。對於HLS，會定期請求分段，因此根據帶寬，可以請求較低質量或較高質量的分段。圖表中一條趨勢呈上坡趨勢（正斜率）的線表示可用比特率正在增加，並且將請求更高質量的分段。相反，圖中的趨勢線呈下坡（負斜率）趨勢，則意味著可用比特率正在下降，並且將請求質量較差的段。

圖中的線對應於視頻可用的演繹形式。當繪製的圖形移動到一條線上方時，玩家將要求更高質量的細分。當繪製的圖形移到一條線以下時，情況正好相反。請注意，當繪圖儀曲線越過一條線時，您很可能不會立即看到視頻質量的變化。所請求的段將必鬚根據請求發送，並且在接收到時附加到當前在緩衝區中的段。
