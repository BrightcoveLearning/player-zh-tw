支持的瀏覽器

強烈建議您定期檢查可用的瀏覽器更新，以確保您擁有最新，最出色的修復程序和性能。

有關受支持的瀏覽器的完整列表，請參見Brightcove Player系統要求文檔。

常見錯誤信息

以下是嘗試播放視頻時可能會收到的錯誤消息列表，以及解決該問題的建議步驟：

MEDIA_ERR_ABORTED

  • 錯誤代碼：1
  • 標題：視頻下載已取消
  • 信息：您中止了媒體播放

可能的原因

瀏覽器出現一些一般性錯誤，表明某些內容停止了其軌道上的視頻下載過程。可能是：

  • 計算機問題，例如崩潰或網絡服務中斷
  • 瀏覽器問題或配置問題
  • 不支持瀏覽器，或當前不支持視頻
  • 當瀏覽器嘗試播放視頻時，防火牆，Web代理或網絡問題阻止了視頻下載

嘗試步驟

以下是嘗試解決此問題的一些步驟：

  • 刷新頁面
  • 嘗試其他瀏覽器

MEDIA_ERR_NETWORK

  • 錯誤代碼：2
  • 標題：視頻連接丟失，請確認您已連接到互聯網
  • 信息：網絡錯誤導致媒體下載部分失敗

可能的原因

以下是一些可能的原因：

  • 防火牆阻止了視頻數據
  • Web代理中斷連接或過濾流量
  • 過濾軟件（例如Sophos或Lightspeed Web Filter阻止內容）
  • 常規網絡故障（例如，與家庭/學校/個人計算機的互聯網連接失敗）

嘗試步驟

以下是嘗試解決此問題的一些步驟：

  • 檢查互聯網在受影響的機器上是否正常運行
  • 刷新頁面，該頁面可以重新啟動可能已斷開的連接
  • 嘗試其他視頻。那樣有用嗎？如果不是，則表明無法整體上獲得內容。

MEDIA_ERR_DECODE

  • 錯誤代碼：3
  • 標題：視頻質量不佳或格式無法在您的瀏覽器上播放
  • 信息：媒體播放由於損壞問題或瀏覽器不支持的媒體使用功能而中止

可能的原因

瀏覽器無法播放請求的內容。這可能在播放期間發生，或更可能在開始時或接近開始時發生。

嘗試步驟

以下是嘗試解決此問題的一些步驟：

  • 刷新頁面，這可能會重新下載由於第一次嘗試而部分下載而導致無法播放的內容。
  • 將您的瀏覽器更新到最新版本。Chrome和Firefox應該提示您更新，但檢查很有用。

MEDIA_ERR_SRC_NOT_SUPPORTED

  • 錯誤代碼：4
  • 標題：該視頻不可用或此瀏覽器不支持
  • 信息：由於服務器或網絡故障或不支持該格式，因此無法加載該介質

可能的原因

瀏覽器由於不可用而無法播放請求的內容，或者無法在此特定瀏覽器中播放。它與錯誤3的不同之處在於，播放在開始之前而不是在播放過程中失敗。

嘗試步驟

在大多數情況下，這兩個設置都是正確的，但可能是您的IT團隊專門設置的。

PLAYER_ERR_TIMEOUT

  • 錯誤代碼： -2
  • 標題：無法下載視頻
  • 信息：無法下載視頻

播放器無法播放內容/緩衝區45秒

可能的原因

  • 帶寬/網絡速度不足
  • 防火牆阻止了視頻數據
  • Web代理中斷連接或過濾流量
  • 過濾軟件（例如Sophos或Lightspeed Web Filter阻止內容）
  • 常規網絡故障（例如，與家庭/學校/個人計算機的互聯網連接失敗）
  • 有時，這可能是由玩家特定的錯誤引起的

嘗試步驟

以下是嘗試解決此問題的一些步驟：

  • 檢查互聯網在受影響的機器上是否正常運行
  • 刷新頁面，該頁面可以重新啟動可能已斷開的連接
  • 嘗試其他視頻。那樣有用嗎？如果不是，則表明無法整體上獲得內容。
  • 問題是否總是在同一時間發生？

播放器調試器

Brightcove Player調試器是一個插件，您可以輕鬆地將其附加到播放器，以捕獲和記錄有關播放器，媒體和廣告的信息。

有關如何設置和使用播放器調試器的詳細步驟，請參見一步步：Brightcove播放器調試器文檔。

捕獲信息

以下信息將有助於為研究您的問題提供支持。

最終用戶信息

收集有關您的系統環境的以下信息：

  • 截取屏幕截圖或導出環境詳細信息http://supportdetails.com/。這些詳細信息包括：
    • 操作系統
    • 屏幕分辨率/瀏覽器尺寸
    • 網絡瀏覽器，包括版本
    • IP地址
  • 截屏或從中導出網絡速度結果http://beta.speedtest.net/

有用的信息

從瀏覽器創建並保存網絡日誌。

以下說明適用於 Chrome，但相同的說明適用於所有受支持的瀏覽器
  • 打開Chrome開發者工具。(F12 on Windows / CMD+OPT+I on Mac)
  • 選擇安慰標籤，以確定是否報告錯誤。
  • 選擇網絡標籤。
  • 右鍵單擊並選擇與內容一起另存為HAR。
  • 拯救.har文件（擴展名為IE的.xml）在您的計算機上，以供進一步檢查。

有關更多詳細說明，請參見創建網絡日誌文檔。

聯繫支持

如果您未通過錯誤消息步驟或播放器調試器解決問題，則可以提交支持請求。

基本故障排除信息

以下信息對於解決您的問題是必不可少的：

  • 受影響內容的URL或視頻名稱
  • 問題發生/開始（如果不是持續發生）的大致時間範圍，我們可以使用該時間範圍來關聯發現/部署/系統問題/已知問題
  • 明確問題的類型（不播放，不加載，暫停和緩衝，質量差，視頻跳轉或剪切，不發出聲音等）
  • 簡短說明，說明如果不是立即顯而易見的話，可以重現此問題的步驟
  • 指出發生問題的配置，以及是否在台式機或移動設備上（操作系統平台，瀏覽器和版本）
  • 在可能的範圍內，確認是否發生了多個/所有用戶以及一個或多個網絡似乎受到影響的情況
  • 提供屏幕截圖或錯誤消息文本（如果適用）
  • 瀏覽器控制台HAR日誌（請參閱上一節中的說明）
  • 錯誤消息中顯示的會話ID。提供會話ID，或者更輕鬆地獲取錯誤消息的屏幕截圖。
