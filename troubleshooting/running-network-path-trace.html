運行網絡路徑跟踪

{{ page.description }}

概述

這traceroute診斷工具會跟踪從您的計算機到相關網站的路徑。該工具通過網絡發送一系列數據包，並從路徑中的每個路由器接收響應。通過這樣做，它確定路徑上是否存在任何問題。

使用支持時，可能會要求您向他們提供跟踪網絡路徑到有關網站的信息。

從macOS或Linux進行跟踪

請按照以下步驟運行網絡路徑跟踪：

  1. 打開終端應用程序的窗口。
  2. 在有問題的網站上輸入以下命令。 
    traceroute your_domain.com

    如果您知道您的網絡使用Internet協議版本6（IPv6），請輸入以下命令：

    traceroute6 your_domain.com
  3. 您應該看到進入您網站的流量路徑。在沿途從路由器收到響應時，它將建立。第一行代表您的路由器。下一行來自您的ISP，並且一直持續到數據包到達您的網站為止。您的輸出應類似於以下內容：
  4. 不用擔心了解輸出。在“終端"窗口中，選擇編輯 > 全選。然後，將輸出複製到剪貼板。
  5. 將輸出粘貼到電子郵件中，然後將其發送給適當的支持人員。

從Windows追踪

請按照以下步驟運行網絡路徑跟踪：

  1. 打開開始菜單並選擇跑步。
  2. 類型cmd然後選擇好的。
  3. 這將打開命令提示符。在有問題的網站上輸入以下命令。 
    tracert your_domain.com

    如果您知道您的網絡使用Internet協議版本6（IPv6），請輸入以下命令：

    tracert -6 your_domain.com
  4. 您應該看到進入您網站的流量路徑。在沿途從路由器收到響應時，它將建立。第一行代表您的路由器。下一行來自您的ISP，並且一直持續到數據包到達您的網站為止。
  5. 不用擔心了解輸出。將輸出複製到剪貼板。
  6. 將輸出粘貼到電子郵件中，然後將其發送給適當的支持人員。
