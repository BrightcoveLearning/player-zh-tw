Brightcove玩家信息模式

{{ page.description }}

查看玩家信息

通過右鍵單擊播放器並選擇來顯示播放器信息。玩家信息，如該屏幕截圖所示。

注意：在移動設備上，您可以使用長按打開模態。

一旦選擇玩家信息您將看到包含有關播放器信息的模式窗口，如此屏幕截圖所示。

注意：僅作為配置一部分的Brightcove提供的插件將出現在外掛程式部分。如果在HTML頁面的JavaScript中添加Brightcove插件，則該插件不會出現在列表中。客戶開發的插件，即使列為播放器配置的一部分，也不會出現。

關閉模式窗口

您可以通過按退出鍵或單擊左上角的關閉圖標。

更改上下文菜單

您可以更改右鍵單擊時出現的菜單，稱為上下文菜單。為此，您必須更改播放器的配置。您可以通過兩種方式執行此操作：

  • 使用Studio的JSON編輯器來自選手們模塊。
  • 執行配置更改的工具是捲曲。

此處顯示了您可以進行的兩個更改以及如何影響它們。

注意：下面的curl語句使用播放器管理API更改播放器配置。在 curl 語句中，您將看到值$EMAIL , $ACCOUNT_ID和$PLAYER_ID .您必須通過這些名稱創建環境變量並分配相應的值，或者為變量提供文字值。

防止菜單出現

如果您希望根本不顯示播放器信息上下文菜單，則可以執行以下任一操作：

使用JSON編輯器

通過添加突出顯示的屬性來更改播放器的配置（請確保保存更改）：

使用捲曲

curl \
  --header "Content-Type: application/json" \
  --user $EMAIL \
  --request PATCH \
  --data '{
      "context_menu": false
  }' \
  https://players.api.brightcove.com/v2/accounts/$ACCOUNT_ID/players/$PLAYER_ID/configuration
注意：每當您使用curl對播放器進行更改時，都需要重新發布該播放器。您可以在Studio中執行此操作，然後會看到您的播放器被標記為需要在播放器中發布地位柱子：

您還可以使用curl來發布播放器：

curl \
  --header "Content-Type: application/json" \
  --user $EMAIL \
  --request POST \
    https://players.api.brightcove.com/v2/accounts/$ACCOUNT_ID/players/$PLAYER_ID/publish

阻止Brightcove菜單選項

如果您想防止關於布萊特科夫從出現菜單選項時，您可以執行以下操作之一：

使用JSON編輯器

通過添加突出顯示的屬性來更改播放器的配置（請確保保存更改）：

使用捲曲

curl \
  --header "Content-Type: application/json" \
  --user $EMAIL \
  --request PATCH \
  --data '{
    "context_menu": {
      "brightcove_branding": false
    }
  }' \
  https://players.api.brightcove.com/v2/accounts/$ACCOUNT_ID/players/$PLAYER_ID/configuration
注意：每當您使用curl對播放器進行更改時，都需要重新發布該播放器。您可以在Studio中執行此操作，然後會看到您的播放器被標記為需要在播放器中發布地位柱子：

您還可以使用curl來發布播放器：

curl \
  --header "Content-Type: application/json" \
  --user $EMAIL \
  --request POST \
  https://players.api.brightcove.com/v2/accounts/$ACCOUNT_ID/players/$PLAYER_ID/publish
